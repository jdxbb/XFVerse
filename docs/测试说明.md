# 测试说明

## 第一轮回归

- 应用可启动，SQLite 自动初始化。
- 左侧导航可切换。
- 设置页 WebDAV 配置、测试连接、扫描路径增删改启停可用。

## 第二轮回归

- 扫描任务页只扫描已启用路径。
- 视频、字幕和其他文件分类正确。
- `MediaFile`、`ScanTaskLog`、`Movie`、`RatingSource`、`SubtitleBinding` 能写入。
- 资源库和详情页能读取真实数据。
- 低置信或失败识别可通过详情页人工修正。

## 第三轮验收

- 浅色 / 深色主题可切换，并能重启保持。
- 首页真实展示最近新增、最近播放、收藏、推荐、统计和扫描状态。
- 详情页可弹出播放器窗口。
- 播放器支持播放、暂停、停止、进度、音量、全屏、播放源切换。
- 字幕优先自动加载同名字幕，并可切换候选字幕。
- 播放记录可保存，重新打开可续播。
- AI 分类、推荐和辅助识别可演示；接口不可用时本地降级。
