# 影片发现 Known Issues

## FD-2.1

- 人物中文搜索依赖 TMDB alias / 本地化数据，不保证稳定；无结果时提示可尝试英文名或原名。
- 地区筛选准确性取决于 TMDB 搜索结果返回的 `origin_country` 等字段；当前首版只对已加载结果做本地二次过滤。
- 未入库影片没有情绪标签时不显示，不通过简介推断。
- OMDb 请求失败会降级为 TMDB 分或暂无评分，不阻塞搜索结果展示。
- 搜索结果的排序和筛选优先作用于已加载结果，不切换到 TMDB discover，以保留 search 相关度。
- 切布局是占位按钮，点击只提示后续接入。
- 榜单仍未实现，进入 FD-2.2。

## FD-2.2

- 榜单排名数字边框和最终视觉样式后续阶段再做。
- 情绪标签仅已入库影片可显示；未入库影片不伪造情绪标签。
- OMDb 请求失败会降级为 TMDB 分或暂无评分，不阻塞榜单展示。
- 海报缓存未接入，仍直接使用远程海报 URL。
- 榜单最多显示前 200 名。
- 滚动加载依赖 WPF `ScrollViewer.ScrollChanged` 接近底部检测；当前未添加加载更多按钮 fallback。
