# 演示说明

## 推荐演示路径

1. 首页：展示影片数量、播放源数量、最近新增、扫描状态和类型分布。
2. 设置页：展示 WebDAV、元数据 API、AI 和主题配置分区。
3. 扫描任务页：手动触发扫描，展示已启用路径和扫描日志。
4. 资源库页：切换海报墙和列表视图，演示搜索、筛选和状态标记。
5. 详情页：展示影片元数据、评分、多播放源、字幕绑定和人工修正入口。
6. 播放器窗口：从详情页弹出播放，演示播放控制、进度、音量、全屏、播放源切换和字幕切换。
7. 猜你喜欢页：展示 AI 推荐理由。

## 答辩重点

- 明确区分 `Movie`、`MediaFile`、`ScanPath`、`SubtitleBinding`。
- 支持 WebDAV 指定路径扫描，不扫描未启用路径。
- 同一影片可绑定多个播放源，并支持默认源。
- 字幕作为独立资源管理，并与具体播放源建立关系。
- 第三轮 UI 使用浅色 / 深色双主题，播放器采用独立弹窗。
