# XFVerse 1.0 软件使用说明书

Last updated: 2026-06-21

适用版本：XFVerse 1.0.0

## 文档定位

本文档说明 XFVerse 的功能、操作流程和操作结果，适合首次使用及日常查阅。

- 安装、升级、卸载和运行环境要求，请参阅《安装说明》。
- 按故障现象组织的排查方法，请使用[XFVerse 帮助中心](help/README.md)。
- 项目结构、开发环境和源码构建方法，由项目 `README.md` 负责。
- 本说明书只描述 XFVerse 1.0 已实现且可从当前界面使用的能力。

## 1. 认识 XFVerse

### 1.1 软件用途

XFVerse 是运行在 Windows 上的桌面媒体库与播放器。它可以：

- 管理本地文件夹和 WebDAV 网盘中的媒体文件。
- 扫描文件并建立软件内的电影、电视剧、季、集和播放源记录。
- 展示影片元数据、评分、演职员、技术信息和海报。
- 播放本地或 WebDAV 媒体，并管理多播放源、音轨和字幕。
- 记录观看进度、观看历史和电影偏好。
- 搜索外部影片信息、查看榜单，并按需使用 AI 推荐和观影画像。

XFVerse 管理的是“软件记录”。除非文档明确说明，否则软件中的隐藏、移出、删除记录或清理缓存不会删除真实媒体文件。

### 1.2 支持的媒体来源

XFVerse 1.0 支持两类来源：

| 来源 | 说明 |
| --- | --- |
| 本地文件夹 | 扫描当前电脑能够访问的磁盘、移动存储或映射位置 |
| WebDAV | 扫描已配置且当前可访问的 WebDAV 服务路径 |

两类来源相互独立。可以只使用其中一种，也可以同时使用。

### 1.3 核心对象

| 对象 | 含义 |
| --- | --- |
| 电影（Movie） | 一部电影的资料、用户状态和一个或多个播放源 |
| 电视剧（Series） | 电视剧的总览资料及其季列表 |
| 季（Season） | 一部电视剧中的一季，承载季级用户状态 |
| 集（Episode） | 某一季中的单集，可拥有一个或多个播放源 |
| 播放源 | 指向真实本地文件或 WebDAV 文件的软件记录 |
| 元数据条目 | 只有影片资料、尚无真实播放文件的记录 |
| 未识别内容 | 已扫描到文件，但证据不足以安全绑定到正式影片的记录 |

同一部电影或同一集可以有多个播放源。例如，本地版本和网盘版本可同时保留，并可指定默认源。

### 1.4 软件记录与真实文件

下列操作的影响不同，执行前应确认操作名称：

| 操作 | 软件中的结果 | 真实本地/WebDAV 文件 |
| --- | --- | --- |
| 移出媒体库 | 从正常媒体库隐藏，可在已移出内容中恢复 | 不删除 |
| 恢复到媒体库 | 重新显示原记录及其已有状态 | 不修改 |
| 删除记录 | 删除相关软件记录、来源关联和相应用户数据 | 不删除 |
| 删除来源路径配置 | 停止使用该扫描配置 | 不删除 |
| 清理缓存 | 清理 XFVerse 管理的缓存文件 | 不删除媒体文件 |

删除记录后，如果对应来源仍启用，后续扫描可能再次发现该文件并重新建立记录。

### 1.5 电影与电视剧的 1.0 边界

- 电影支持观看状态、喜爱、想看、不想看、AI 推荐和观影洞察。
- 电视剧以 Series、Season、Episode 组织。
- 电视剧的喜爱、想看和不想看状态记录在 Season 层级。
- Episode 支持已看状态和播放历史。
- XFVerse 1.0 的 AI 推荐、观影画像和观影统计输入仅使用电影数据，不把电视剧计入这些能力。

## 2. 界面与导航

### 2.1 主窗口

主窗口由侧边导航、顶部区域和当前页面组成。侧边栏可展开或收起，顶部提供主题切换和用户菜单。

主导航包括：

| 页面 | 用途 |
| --- | --- |
| 首页 | 查看统计摘要、继续观看、最近添加和推荐预览 |
| 媒体库 | 搜索、筛选、浏览和批量管理已建立的媒体记录 |
| 影片发现 | 搜索外部影片资料、查看榜单和使用 AI 推荐 |
| 观影历史 | 按时间查看实际播放产生的历史记录 |
| 收藏夹 | 集中查看“喜爱”和“想看”内容 |
| 观影洞察 | 查看电影观影统计和 AI 画像 |

扫描任务和设置通过用户菜单进入。电影详情、电视剧总览、季详情、集详情和播放器会在相应操作中打开，不作为固定主导航项显示。

### 2.2 首页

首页用于快速继续日常操作，可能包含：

- 媒体库统计摘要。
- 继续观看。
- 最近观看。
- 最近添加。
- 媒体库预览。
- AI 推荐预览及“发现更多影片”入口。

首页内容取决于当前数据库、播放历史和外部服务配置。新安装或尚未扫描时，部分区域为空是正常现象。

### 2.3 用户菜单

用户菜单提供：

- 个人资料：编辑仅保存在本机的昵称、账号展示信息、签名、性别、年龄、邮箱和头像。
- 扫描任务：配置扫描来源并执行扫描。
- 设置：配置通用选项、WebDAV 和外部服务。
- 退出登录：1.0 中不代表云账号退出，属于本地用户界面的保留入口。

个人资料没有云同步能力。不要把它当作在线账号系统使用。

### 2.4 返回与详情导航

从媒体卡片、列表项、历史记录、收藏条目或发现结果进入详情后，可使用页面提供的返回入口回到上一层。电视剧通常按以下层级浏览：

`电视剧总览 → 季详情 → 集详情 → 播放器`

## 3. 首次使用推荐流程

首次使用建议按以下顺序完成：

1. 启动 XFVerse，按偏好选择主题和关闭窗口行为。
2. 从用户菜单进入“扫描任务”。
3. 添加至少一个本地文件夹，或先在“设置”中配置 WebDAV。
4. 如需在线影片资料，在“设置 → API 配置”中配置 TMDB。
5. 如需在线字幕，配置 OpenSubtitles。
6. 如需 AI 推荐、AI 辅助识别或观影画像，配置兼容的 AI 服务。
7. 返回“扫描任务”，分别执行本地扫描或 WebDAV 扫描。
8. 查看扫描进度、结果计数和近期日志。
9. 进入“媒体库”，检查电影、电视剧和未识别内容。
10. 对识别不准确或证据不足的条目执行人工修正。
11. 打开详情页，选择默认播放源并开始播放。
12. 使用收藏夹、观影历史和观影洞察管理后续内容。

TMDB、OpenSubtitles、OMDb 和 AI 都是按功能使用的外部服务。未配置某项服务时，只影响依赖它的能力，不会使本地媒体库和基础播放整体失效。

## 4. 添加和管理媒体来源

### 4.1 添加本地文件夹

1. 打开用户菜单中的“扫描任务”。
2. 找到本地来源区域。
3. 使用文件夹选择器添加目录，或按界面提供的手动方式填写目录。
4. 确认路径已启用。
5. 可直接运行该路径，或执行全部本地扫描。

注意：

- 路径必须是当前 Windows 用户有权访问的目录。
- 移动磁盘或网络映射位置离线时，本次扫描可能失败或无法确认文件状态。
- 不要重复添加相同目录或互相包含的多层目录，以免造成重复扫描。
- 修改或删除路径配置只影响 XFVerse 的扫描配置，不会删除目录中的文件。

### 4.2 配置 WebDAV

1. 打开“设置 → API 配置”。
2. 填写 WebDAV 服务地址、用户名和密码。
3. 保存连接信息。
4. 使用“测试连接”确认服务可访问。
5. 添加一个或多个需要扫描的 WebDAV 路径，并确认所需路径已启用。
6. 返回“扫描任务”，执行 WebDAV 扫描。

WebDAV 地址、账号和密码应以服务提供方给出的信息为准。测试失败时先检查：

- 服务地址是否完整且可访问。
- 当前网络是否能连接该服务。
- 账号是否有列出目录和读取文件的权限。
- 扫描路径是否存在。
- 代理、防火墙或证书环境是否阻止连接。

凭据只用于本机配置，不应写入截图、日志说明或共享文档。

### 4.3 启用、停用、编辑和删除路径

- 启用：允许后续扫描处理该路径。
- 停用：保留配置，但常规扫描不处理该路径。
- 编辑：修改软件中的路径配置。
- 删除：删除软件中的路径配置。

上述操作都不会删除真实文件。已经建立的媒体记录如何显示，取决于后续扫描结果和媒体库管理操作。

### 4.4 本地与 WebDAV 分开扫描

本地扫描和 WebDAV 扫描是不同命令：

- 只修改本地文件后，可仅运行本地扫描。
- 只修改网盘内容后，可仅运行 WebDAV 扫描。
- 本地路径还可以按单个路径运行。
- 扫描进行中可使用取消操作；取消只停止当前任务，不删除已存在的真实文件。

### 4.5 启动时自动扫描

设置中可启用启动时自动执行 WebDAV 扫描。启用后应确保：

- 启动时网络已可用。
- WebDAV 服务响应正常。
- 已保存有效连接和至少一个启用路径。

该选项只针对 WebDAV，不等同于自动扫描所有本地目录。

## 5. 扫描与识别

### 5.1 查看扫描过程

扫描任务页面会显示当前进度、扫描记录、忽略数、错误数、耗时和近期日志。判断扫描是否成功时，不应只看已绑定影片数量，还应同时检查：

- 实际扫描文件数。
- 被忽略项目及原因。
- 错误数和错误摘要。
- 未识别或待修正项目。
- 媒体库中新增、恢复或更新的记录。

### 5.2 自动识别的处理原则

XFVerse 会结合文件名、目录结构、本地规则和已配置的外部服务尝试识别媒体。AI 结果只作为受控提示，不会无条件替代本地规则、TMDB 结果或用户确认。

当证据不足时，软件可能：

- 建立未识别内容。
- 标记为待修正。
- 保留候选结果等待人工确认。
- 将疑似电视剧内容组织为未识别季。

这种结果表示软件避免了高置信错误绑定，不代表文件扫描失败。

### 5.3 电影、电视剧和其他内容

- 电影会进入电影详情结构。
- 可识别的电视剧会按 Series、Season、Episode 组织。
- 无法安全归类的内容会保留在“其他”或未识别结构中。
- 只有真实发现的文件才会建立播放源；外部元数据搜索不会虚构不存在的媒体文件。

### 5.4 技术信息探测

扫描或详情页可以探测分辨率、时长、码率等技术信息。探测失败时：

- 原有媒体记录会保留。
- 播放源不会因为一次探测失败而被自动删除。
- 可在来源恢复可访问后重试。

### 5.5 重新扫描

重新扫描用于发现新增文件、更新现有来源状态，并处理暂时离线后重新出现的来源。需要注意：

- 同一来源路径恢复访问后，可通过重新扫描恢复其可用状态。
- 已删除的软件记录在来源仍存在时可能被重新发现。
- 重新扫描不会创建指向不存在文件的播放源。
- 识别结果仍应在媒体库中人工抽查，尤其是命名不规范的内容。

扫描无结果、识别错误或电视剧分组异常时，见[扫描与识别问题](help/扫描与识别.md)。

## 6. 使用媒体库

### 6.1 内容分类

媒体库支持查看：

- 全部。
- 电影。
- 电视剧。
- 其他。

“其他”通常包含无法安全识别为正式电影或电视剧的已扫描内容。

### 6.2 搜索、筛选和排序

媒体库提供关键词搜索，并可组合使用以下筛选：

- 来源：本地、网盘。
- 内容类型：电影、电视剧、其他。
- 年代。
- 已看、未看。
- 喜爱、想看、不想看及其他状态。
- 类型、情绪和场景标签。
- 有播放源、无播放源。

支持按最近更新、标题、年份或评分排序，并可切换排序方向。筛选组合过多导致空结果时，可逐项清除条件。

### 6.3 海报与列表视图

- 海报视图适合按封面浏览。
- 列表视图适合集中查看文字信息。

切换视图不会修改媒体数据。

### 6.4 打开条目

- 电影条目打开电影详情。
- 电视剧条目打开电视剧总览或相应季结构。
- 未识别内容可能只能进入可用的修正或来源管理流程。
- 没有播放源的元数据条目仍可查看资料，但不能直接播放。

### 6.5 批量选择

进入选择模式后，可以：

- 选择当前可见项目。
- 清除选择。
- 批量标记已看或未看。
- 对适用项目执行 AI 辅助识别。
- 移出媒体库。
- 删除软件记录。
- 对未识别电视剧内容执行人工聚合或聚合后识别。

批量操作只作用于当前已选择项目。执行高影响操作前应再次核对筛选条件和选择数量。

### 6.6 移出媒体库

“移出媒体库”是可恢复的隐藏操作：

- 条目从正常媒体库隐藏。
- 不删除元数据。
- 不清除想看、喜爱、不想看、已看等用户状态。
- 不删除观看历史。
- 不删除本地或 WebDAV 文件。
- 不停用来源路径。

可在已移出内容区域恢复单个条目或相关分组。

### 6.7 删除记录

“删除记录”用于清除 XFVerse 内的相关软件数据：

- 删除媒体及来源关联记录。
- 清除关联的观看历史和收藏状态。
- 不删除本地文件。
- 不删除 WebDAV 文件。

如果文件仍位于启用的扫描路径，后续扫描可能重新建立记录。要避免重新发现，应同时调整扫描范围或停用相应路径，但这仍不会删除真实文件。

移出、恢复、删除记录或批量操作失败时，见[媒体库与数据操作问题](help/媒体库与数据操作.md)。

### 6.8 已移出内容

已移出内容区域支持：

- 恢复单个项目。
- 恢复相关分组。
- 删除单个软件记录。
- 删除相关分组的软件记录。
- 查看可用详情。

电视剧总览本身不一定具有独立的隐藏状态；恢复时可能需要恢复其季或来源记录。

## 7. 电影与电视剧详情

### 7.1 电影详情

电影详情页用于查看和管理：

- 标题、简介、海报和背景图。
- 类型、上映信息、演职员。
- 外部评分和本地技术信息。
- 用户状态。
- 一个或多个播放源。
- 识别修正入口。

不同电影的数据完整度可能不同。外部服务未配置、影片尚未匹配或远端服务暂时不可用时，部分元数据区域可能为空。

### 7.2 播放电影

有播放源时，可以：

- 播放默认源。
- 直接播放某个指定源。
- 将某个来源设为默认源。
- 对来源重新探测技术信息。

没有播放源的电影属于元数据条目，只能查看资料和设置适用的用户状态。XFVerse 不会为这类条目创建虚假文件。

### 7.3 管理电影播放源

同一电影下的多个来源可能来自不同文件、清晰度或存储位置。来源列表会尽量展示来源类型和已探测的技术信息。

可用操作包括：

- 设置默认来源。
- 播放指定来源。
- 重新探测来源。
- 将错误聚合的来源拆分出来，并重新进入识别流程。

“拆分”只调整 XFVerse 中的识别和关联记录，不会分割、移动或删除真实视频文件。

### 7.4 修正电影识别

发现电影识别错误时，可从详情或来源操作进入修正流程。修正时可以：

- 搜索并确认正确的 TMDB 电影。
- 将来源改为某部电视剧的具体集。
- 将来源加入已有的未识别季。
- 使用 AI 辅助生成搜索提示。

AI 辅助只提供候选，不代表自动确认。最终目标应由用户核对标题、年份、季号和集号后确认。修正失败时，软件应保留原记录或回滚本次变更，而不是删除真实文件。

### 7.5 电视剧总览

电视剧总览展示 Series 级资料和季列表。选择某一季后进入季详情。

可能存在以下情况：

- 已识别的正式季。
- 仅有外部资料、尚无本地播放源的季。
- 由扫描结果形成的未识别季或待修正季。

未识别季复用季级界面是为了集中管理疑似剧集文件，不表示它已经被确认绑定到正式电视剧。

### 7.6 季详情

季详情用于：

- 查看季级资料和集列表。
- 播放某一集。
- 标记单集已看或未看。
- 标记整季已看或未看。
- 设置季级喜爱、想看或不想看状态。
- 进入季识别修正。

季级状态的约束与电影状态类似：

- 喜爱只适用于已看的季。
- 已看的季不再使用“想看”表达。
- 不想看与喜爱、想看互斥。

标记整季状态会影响该季中的单集观看状态。执行前应确认当前季和集列表无误。

### 7.7 集详情

集详情可展示：

- 集标题、简介、播出信息和所属季。
- 当前已看状态。
- 一个或多个播放源。
- 默认播放源。
- 技术信息和来源操作。

集详情支持播放指定来源、设置默认来源、重新探测、拆分错误来源和识别修正。没有来源的集仍可作为元数据条目显示，但不能播放。

### 7.8 上一集与下一集

播放电视剧集时，播放器可以在当前季的有效集顺序中切换上一集或下一集。电影播放不提供集切换语义。

如果相邻集没有可用来源，或当前条目没有可靠的季集关系，相应按钮可能不可用。

### 7.9 修正电视剧和集

电视剧相关修正可以把来源移动到：

- 已识别电视剧的指定季和指定集。
- 已有未识别季中的指定位置。
- 其他经用户确认的有效目标。

当目标集已有来源时，新来源作为该集的额外播放源保留，而不是创建空白集或覆盖真实文件。修正前应核对季号、集号和目标剧名。

## 8. 使用播放器

### 8.1 打开播放器

可从以下位置开始播放：

- 首页的继续观看或最近观看。
- 媒体库的可播放条目。
- 电影详情中的默认源或指定源。
- 季详情或集详情中的剧集来源。
- 观影历史中的有效条目。

来源离线、权限不足或远端网络不可用时，播放器可能无法打开。此时应先确认来源仍可访问。

### 8.2 基本控制

播放器提供：

- 播放和暂停。
- 进度跳转。
- 音量调整和静音。
- 窗口与全屏切换。
- 电视剧上一集和下一集。
- 播放源切换。
- 音轨切换。
- 字幕选择。

拖动进度时，最终可跳转位置还取决于媒体格式、来源响应和播放器底层能力。

### 8.3 键盘快捷键

| 按键 | 功能 |
| --- | --- |
| `Space` | 播放或暂停 |
| `F` | 切换全屏 |
| `M` | 静音或恢复声音 |
| `Left` | 后退 5 秒 |
| `Right` | 前进 5 秒 |
| `Ctrl + Left` | 后退 30 秒 |
| `Ctrl + Right` | 前进 30 秒 |
| `Up` | 音量增加 5 |
| `Down` | 音量降低 5 |
| `Esc` | 退出全屏 |

双击视频区域也可切换全屏。

### 8.4 音量

播放器音量范围为 0—200：

- 0 表示无声。
- 100 可视为常规满音量。
- 高于 100 属于额外增益，可能产生失真或爆音。

使用耳机时不要直接设置过高音量。

### 8.5 切换播放源

当同一电影或剧集有多个来源时，可在来源菜单中查看并切换。菜单可能展示：

- 本地或 WebDAV 来源类型。
- 分辨率。
- 码率。
- 文件大小。
- 默认来源标识。
- 可用的续播信息。

切换来源后，播放器会尝试在新来源上继续当前内容。不同版本时长不一致时，续播位置可能需要手动调整。

### 8.6 音轨

音轨菜单会列出播放器从当前媒体中检测到的音轨。可选择所需语言或声道。

音轨名称和语言取决于文件内的封装信息；文件没有正确标记时，界面可能只能显示通用名称。

### 8.7 字幕类型

字幕菜单按来源区分：

- 无字幕。
- 嵌入字幕：封装在当前视频文件中。
- 外挂字幕：来自本地或已关联的独立字幕文件。
- 在线字幕：通过在线字幕搜索下载并绑定。

切换播放源后，嵌入字幕集合可能变化，因为它属于具体视频文件。

### 8.8 搜索和应用在线字幕

使用在线字幕前，应在“设置 → API 配置”中启用并配置 OpenSubtitles。

基本流程：

1. 播放目标电影或剧集。
2. 打开字幕菜单。
3. 选择在线字幕搜索。
4. 核对搜索类型、标题或关键词和语言。
5. 按结果信息选择字幕。
6. 下载并应用。

打开搜索对话框时，播放器会暂停。搜索结果质量由关键词、影片匹配和外部服务返回决定。

在线字幕下载后可以形成字幕绑定。删除绑定与清理孤立字幕缓存是不同操作：

- 删除绑定：解除该字幕与当前媒体的关联。
- 移除临时字幕：停止使用对应临时内容。
- 清理孤立缓存：清除 XFVerse 判断为不再被引用的在线字幕缓存。

播放器、音轨或字幕出现错误时，见[播放与字幕问题](help/播放与字幕.md)。

### 8.9 续播与观看历史

播放过程中，XFVerse 会根据实际播放情况保存进度并写入历史。再次打开同一内容时可尝试继续播放。

以下情况可能影响续播：

- 播放源已更换且时长不同。
- 原来源不可访问。
- 对应软件记录已删除。
- 历史或进度数据已被清除。

正常关闭播放器会保存当前可用进度。不要通过强制结束进程来替代正常关闭。

### 8.10 WebDAV 播放与缓存

WebDAV 播放依赖网络和远端服务响应。如果 XFVerse 已有完整可用缓存，播放器可优先使用该缓存；没有缓存时仍会尝试正常远端播放。

缓存状态属于辅助信息，不表示真实网盘文件已被移动或复制到媒体库。清理 XFVerse 缓存不会删除 WebDAV 上的媒体文件。

## 9. 影片发现

### 9.1 功能边界

影片发现用于浏览外部影片资料，不等于扫描真实文件。通过发现功能获得的条目可以只有元数据，没有播放源。

该页面包含：

- 影片搜索。
- 榜单。
- AI 推荐。

常规搜索和榜单依赖 TMDB。AI 推荐还需要已配置的 AI 服务。

### 9.2 搜索影片

搜索支持电影和电视剧，并可按标题或人物等条件查找。可用筛选根据当前页面状态组合，包括：

- 媒体类型。
- 地区和语言。
- 年代。
- 观看状态。
- 是否有来源。
- 是否已在媒体库。
- 收藏状态。
- 排序和分页。

支持海报和列表方式浏览。搜索条件过严时，应减少筛选项或更换关键词。

### 9.3 榜单

榜单可按电影或电视剧查看外部热门、高分或趋势内容，并支持不同时间范围和分页。

榜单数据来自外部服务，内容、排序和更新时间不由 XFVerse 单独决定。

### 9.4 元数据条目

从发现页添加或标记的外部内容可能只有影片资料：

- 可以在适用范围内设置想看或其他用户状态。
- 可以进入媒体库资料结构。
- 没有真实文件时不能播放。
- 软件不会因此创建不存在的 `MediaFile` 或伪造播放源。

### 9.5 AI 推荐

AI 推荐 1.0 仅面向电影。可根据当前电影状态和自定义偏好生成推荐，并按来源、观看状态等范围筛选。

可执行的操作包括：

- 刷新推荐批次。
- 调整自定义偏好。
- 将电影标记为想看。
- 将电影标记为不想看。
- 打开可用详情。

AI 输出可能使用缓存，也可能在服务不可用时进入降级或错误状态。推荐理由是辅助信息，不保证影片一定适合用户，也不会自动确认影片识别结果。

## 10. 用户状态与个人内容

### 10.1 电影状态

电影支持：

- 已看或未看。
- 喜爱。
- 想看。
- 不想看。

状态之间存在约束：

- 电影标记为已看后，会清除“想看”。
- 电影改为未看后，会清除“喜爱”。
- “喜爱”只适用于已看的电影。
- 已看的电影不使用“想看”；可改用“喜爱”表达偏好。
- “不想看”与“喜爱”“想看”互斥。

这些规则用于避免同一电影同时出现逻辑冲突的状态。

### 10.2 电视剧状态

- Series 用于组织电视剧整体资料。
- 喜爱、想看和不想看记录在 Season。
- 已看状态可以记录到具体 Episode，也可以从季详情批量设置整季。
- Episode 不承载电影级的喜爱或想看状态。

### 10.3 收藏夹

收藏夹包含两个主要页签：

- 喜爱。
- 想看。

其中可以出现适用的电影和电视剧季。执行“移除”时，只取消当前收藏状态，不会删除媒体记录、播放源或真实文件。

### 10.4 观影历史

观影历史来自实际播放行为，可按以下范围查看：

- 全部。
- 今天。
- 本周。
- 本月。
- 指定日期。

有效记录可以打开电影详情或集详情。如果关联媒体记录已被删除，页面会安全处理该记录，而不会创建虚假来源。

### 10.5 个人资料

个人资料可编辑：

- 用户名和本地展示账号。
- 手机号。
- 个性签名。
- 性别和年龄。
- 邮箱。
- 头像。

这些信息只保存在本机，不提供账号注册、云端同步或跨设备恢复。签名长度受界面限制，且不支持换行。

## 11. 观影洞察

### 11.1 功能范围

观影洞察包含：

- 画像分析。
- 观影统计。

XFVerse 1.0 只使用电影数据生成洞察。电视剧季、剧集观看和电视剧状态不会进入统计、画像或 AI 推荐输入。

### 11.2 观影统计

统计页可在“本月”和“全部”等范围间切换，并通过日历查看不同日期。统计内容可能包括：

- 观看时长。
- 观看天数。
- 类型和标签偏好。
- 常用观看时段。
- 片长偏好。
- 偏好组合。

从日历选择日期后，可进入对应日期的观影历史。统计结果来自 XFVerse 已保存的电影播放记录，不等同于磁盘中所有影片的总时长。

### 11.3 画像分析

画像分析使用已保存的电影观看数据和 AI 服务生成描述性结果。页面可能处于：

- 正常生成或已生成。
- 使用已有缓存。
- 数据不足。
- AI 未配置。
- 外部服务失败。

当刷新失败但已有可用快照时，页面可能继续展示旧快照。画像属于辅助性总结，不应作为身份、健康或专业判断。

### 11.4 让统计更准确

- 通过播放器正常观看，使进度和历史得到保存。
- 不要把仅打开详情页视为已经观看。
- 修正错误识别，避免电影被关联到错误资料。
- 删除记录会同时影响相应历史和状态。
- 电视剧不计入 1.0 电影洞察，这是产品边界，不是统计遗漏。

## 12. 设置

### 12.1 通用设置

通用设置用于调整桌面应用行为、缓存和关于信息。

主要选项包括：

- 主题：跟随系统、浅色、深色。
- 关闭主窗口时缩小到托盘或退出软件。
- 点击播放后自动进入全屏。
- 下次启动时自动扫描 WebDAV。
- 软件缓存状态和清理。
- 版本和关于信息。

### 12.2 关闭窗口行为

可选择：

- 缩小到托盘：关闭主窗口后 XFVerse 仍在后台运行，可从系统托盘恢复或退出。
- 退出软件：关闭主窗口后结束 XFVerse。

如果选择缩小到托盘，看到主窗口消失不代表程序已经退出。安装、升级或备份数据库前，应从托盘明确退出。

### 12.3 自动全屏

启用“点击播放后自动全屏”后，新打开的播放器默认进入全屏。该设置不改变媒体内容，也不影响键盘 `F` 和双击切换全屏。

### 12.4 缓存管理

设置页会显示和管理以下软件缓存：

- 海报缓存。
- 其他缓存。
- 在线字幕缓存。

海报缓存支持设置容量上限，并可清理后按需重新生成。

清理其他缓存只处理 XFVerse 标记为可清理的缓存记录，不删除电影、电视剧、用户状态、观看进度或真实媒体文件。

在线字幕清理只删除未被 Movie、Episode 或当前播放源绑定引用的孤立缓存。仍有绑定的字幕、扫描发现的外挂字幕和真实媒体不会被删除。

### 12.5 WebDAV 设置

WebDAV 区域用于：

- 保存连接地址和凭据。
- 测试连接。
- 管理 WebDAV 扫描路径。

保存配置不等于已完成扫描。连接和路径配置完成后，仍需前往扫描任务执行 WebDAV 扫描，或启用启动时自动扫描。

### 12.6 TMDB

TMDB 用于：

- 电影和电视剧在线搜索。
- 榜单。
- 扫描识别和人工修正中的元数据匹配。
- 海报、简介、演职员等在线资料。

按界面要求填写 TMDB 凭据并执行测试。TMDB 不可用时，本地文件扫描和基础播放仍可工作，但在线发现和元数据识别能力会受到影响。

### 12.7 OMDb

OMDb 用于补充适用的外部影片数据或评分。它不是本地播放的必要条件，也不能替代 TMDB 承担全部搜索和识别功能。

### 12.8 OpenSubtitles

OpenSubtitles 用于在线字幕搜索和下载。配置包括：

- API Key。
- 可选账号和密码。
- 默认字幕语言。
- 是否启用在线字幕能力。

保存后应执行测试。只有 API Key 时也可以按界面允许的能力保存并测试；具体可用结果由外部服务账号权限和接口响应决定。

### 12.9 AI 服务

AI 设置用于兼容接口的服务地址、密钥、默认模型和不同任务的模型路由。任务可能包括：

- 扫描电影标签。
- 电视剧不确定项和完整识别。
- 详情修正。
- 批量 AI 辅助识别。
- 电影 AI 推荐。
- 电影观影画像。

建议流程：

1. 填写服务地址和密钥。
2. 保存配置。
3. 测试服务并获取或确认模型。
4. 选择默认模型。
5. 按需设置各任务模型。
6. 再进入对应功能使用。

AI 服务未配置时，本地媒体库、基础扫描规则和播放仍可使用。AI 失败时不要把生成内容当作已确认元数据。

外部服务测试、认证、额度或模型异常时，见[影片发现与外部服务问题](help/影片发现与外部服务.md)。

### 12.10 凭据保存

WebDAV 和外部服务凭据保存在当前 Windows 用户的软件数据中，并使用当前用户上下文的本机保护机制处理敏感值。

限制如下：

- 凭据不提供云端同步。
- 复制数据库或配置到其他电脑，不保证凭据可以直接恢复使用。
- 不要公开用户数据目录、配置文件或未经脱敏的日志。
- 更换 Windows 用户或系统后，应准备重新输入外部服务凭据。

### 12.11 版本与关于

关于区域显示 XFVerse 名称、当前软件版本和产品说明，并可提供官方网站入口。提交问题时应记录这里显示的版本和安装包架构。

## 13. 数据管理与安全

### 13.1 用户数据位置

默认用户数据目录为：

```text
%LOCALAPPDATA%\MediaLibrary
```

内部目录名 `MediaLibrary` 为兼容既有数据而保留。主要内容包括：

| 内容 | 默认名称 |
| --- | --- |
| SQLite 数据库 | `media-library.db` |
| 视频缓存 | `VideoCache` |
| 海报缓存 | `PosterCache` |
| 在线字幕缓存 | `OnlineSubtitles` |
| 诊断日志 | `Logs` |
| 本地资料和偏好 | 用户资料、头像、JSON 配置及数据库设置 |

用户数据目录与安装目录分开。默认卸载不会自动删除该目录。

### 13.2 备份

建议在升级、架构切换、重装系统或执行高影响数据处理前备份整个用户数据目录。

备份步骤：

1. 正常退出 XFVerse，包括托盘中的后台实例。
2. 确认软件已不再运行。
3. 复制整个 `%LOCALAPPDATA%\MediaLibrary` 目录到安全位置。
4. 保留备份的版本、日期和对应 XFVerse 版本说明。

不要只在应用运行时复制数据库文件，否则可能得到不一致的数据库快照。

### 13.3 恢复原则

恢复前应：

1. 退出 XFVerse。
2. 先备份当前用户数据目录。
3. 使用与备份兼容的软件版本。
4. 恢复完整目录，而不是随意拼接不同时间的数据库和配置。
5. 启动后检查媒体库、设置、历史和来源状态。

恢复用户数据不会自动恢复已经删除、移动或离线的真实媒体文件。

备份、恢复、数据库或缓存问题见[缓存、备份与恢复问题](help/缓存备份与恢复.md)。

### 13.4 卸载后的数据

默认卸载主要删除程序文件和安装器创建的快捷方式，保留用户数据库、设置、状态、历史和缓存。

如果确实需要完全清除 XFVerse 软件数据，应先备份，然后在卸载并确认不再需要数据后手动删除用户数据目录。该操作不会删除用户自行存放在其他位置的本地影片、字幕或 WebDAV 文件。

### 13.5 隐私

XFVerse 会在本机处理媒体路径、观看历史、用户状态、外部服务配置和日志。分享问题信息时：

- 不提交完整本地路径。
- 不提交完整 WebDAV 地址。
- 不提交账号、密码、Token、API Key。
- 不直接提交数据库。
- 不提交未经检查的完整日志或私人媒体清单。
- 优先提供脱敏错误摘要、版本、架构和有限日志片段。

### 13.6 外部服务数据

使用 TMDB、OMDb、OpenSubtitles、WebDAV 或 AI 时，相关查询会发送到对应外部服务。是否使用、发送范围、账号权限和服务条款由用户自行配置和确认。

不需要某项在线能力时，可以不配置相应服务。

## 附录 B：功能与依赖

| 功能 | 必需条件 | 未满足时的影响 |
| --- | --- | --- |
| 本地扫描 | 可访问的本地目录 | 无法扫描对应目录 |
| WebDAV 扫描 | 网络、有效连接和启用路径 | 只影响 WebDAV 来源 |
| 本地基础播放 | 可访问媒体文件 | 对应来源无法播放 |
| WebDAV 播放 | 网络、权限和可用远端文件 | 对应远端来源无法播放 |
| 影片搜索和榜单 | TMDB 配置及网络 | 影片发现在线数据不可用 |
| 在线识别和修正 | TMDB 及适用配置 | 只能使用本地规则或保留待修正 |
| OMDb 补充数据 | OMDb 配置及网络 | 缺少对应补充数据 |
| 在线字幕 | OpenSubtitles 配置及网络 | 仍可用嵌入或本地外挂字幕 |
| AI 辅助识别 | AI 配置及网络 | 不影响本地规则和人工修正 |
| AI 推荐 | AI 配置、电影数据及网络 | 推荐页显示未配置、降级或错误状态 |
| 观影画像 | AI 配置和足够电影历史 | 可显示数据不足、未配置或旧缓存 |
| 观影统计 | 已保存的电影观看记录 | 数据为空或不完整 |

## 附录 C：术语

| 术语 | 说明 |
| --- | --- |
| 媒体记录 | XFVerse 数据库中的电影、季、集或未识别内容 |
| 播放源 | 指向真实本地或 WebDAV 文件的软件关联 |
| 元数据 | 标题、简介、海报、演职员、评分等影片资料 |
| 默认源 | 打开内容时优先使用的播放源 |
| 未识别季 | 尚未确认正式剧名或季信息的电视剧式文件分组 |
| 待修正 | 需要用户核对和确认识别结果的状态 |
| 嵌入字幕 | 封装在视频文件内部的字幕轨 |
| 外挂字幕 | 独立字幕文件 |
| 在线字幕绑定 | 下载字幕与电影、集或来源之间的软件关联 |
| 移出媒体库 | 隐藏但保留记录和用户状态 |
| 删除记录 | 删除软件数据但不删除真实媒体文件 |
| 缓存 | XFVerse 为性能、在线数据、海报、字幕或播放临时保存的数据 |
