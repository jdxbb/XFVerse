# 播放与字幕问题

Last updated: 2026-06-21

## 适用范围

本文处理播放器无法打开、黑屏、卡顿、WebDAV 跳转、音轨、嵌入字幕、外挂字幕、在线字幕、续播和多播放源问题。

播放器正常操作见[软件使用说明书第 8 章](../使用说明书.md)。

## 1. 点击播放没有反应

### 常见原因

- 当前条目没有真实播放源。
- 默认播放源已离线或被删除。
- 本地文件不可访问。
- WebDAV 网络或认证失败。
- 播放引擎原生文件缺失或架构不匹配。

### 检查步骤

1. 打开详情页查看播放源列表。
2. 确认至少一个来源可用。
3. 有多个来源时，尝试播放另一个来源。
4. 本地来源用文件资源管理器确认文件仍存在。
5. WebDAV 来源先测试连接和网络。
6. 记录播放器或详情页显示的错误摘要。

### 修复步骤

- 没有来源：重新扫描真实文件，或保留为元数据条目。
- 默认源失效：选择可用来源并设为默认。
- 程序文件损坏：运行同版本修复安装。
- 架构错误：安装与 Windows 匹配的 x64 或 ARM64 包。

不要手工构造不存在的播放源。

## 2. 播放器打开后黑屏

### 先判断是否仍有声音

- 有声音无画面：更可能是视频解码、渲染或编码兼容问题。
- 无声音无画面：更可能是来源、网络或播放引擎加载失败。

### 检查步骤

1. 暂停后继续播放。
2. 退出全屏后观察窗口状态。
3. 尝试同一影片的其他播放源。
4. 尝试一个已知可播放的普通 H.264 本地视频。
5. 检查当前文件是否为 HEVC、4K、HDR 或异常封装。
6. 查看问题时间附近的 `mpv-playback.log`。

### 修复方向

- 只有单个文件失败：可能是文件编码或封装问题。
- 所有本地视频失败：修复安装并核对架构。
- 只有 WebDAV 失败：检查远端服务、Range 和网络。
- HEVC 4K 失败：记录硬件、编码和日志，使用其他来源验证。

不要用删除媒体记录来修复黑屏。

## 3. 播放器显示来源不可用

### 本地来源

检查：

- 文件是否移动、重命名或删除。
- 移动磁盘是否连接。
- 当前 Windows 用户是否仍有权限。

恢复原路径后重新扫描，XFVerse 会尝试重新关联。

### WebDAV 来源

检查：

- 网络是否可用。
- WebDAV 凭据是否仍有效。
- 远端文件是否移动。
- 当前账号是否有读取权限。

来源恢复后执行 WebDAV 重扫。不要先删除原记录。

## 4. WebDAV 播放卡顿或跳转失败

### 常见原因

- 网络带宽或延迟不足。
- 服务端不稳定。
- 服务端不正确支持 HTTP Range。
- 大文件随机跳转需要重新缓冲。
- 代理或网关改变了请求行为。

### 检查步骤

1. 暂停片刻等待缓冲后继续。
2. 从头播放，判断是否只有跳转失败。
3. 尝试较低码率或较小的来源。
4. 在同一网络中确认 WebDAV 服务稳定。
5. 检查服务是否支持 Range 读取。
6. 对比本地来源播放结果。

### 关于大体积文件

70GB 以上 WebDAV 文件和高码率内容可能在部分网络或服务中卡顿。XFVerse 1.0 的最终表现仍需 Phase 8.9 RC 记录，不应把单一网络环境的结果泛化。

### 不要做什么

- 不要连续快速拖动进度条。
- 不要把关闭防火墙作为长期解决方案。
- 不要清空数据库来解决网络卡顿。

## 5. WebDAV 服务是否必须支持 Range

远端播放和进度跳转通常依赖服务端正确处理范围请求。Range 支持不完整时可能出现：

- 无法从中间开始播放。
- 拖动后回到开头。
- 长时间缓冲。
- 续播位置无法快速定位。

应向 WebDAV 服务提供方确认 Range 能力，或使用本地来源进行对照。

## 6. 播放一段时间后停止或报错

### 检查步骤

1. 记录停止时间点。
2. 重新打开并从附近位置播放。
3. 判断错误是否固定发生在同一时间点。
4. 检查文件是否损坏或远端连接是否中断。
5. 检查磁盘空间和 XFVerse 缓存状态。

固定时间点失败通常更接近文件或封装问题；随机失败更可能与网络、来源或系统负载有关。

## 7. 没有声音

### 检查步骤

1. 确认播放器未静音，可按 `M` 切换。
2. 检查 XFVerse 音量。
3. 检查 Windows 音量混合器。
4. 打开音轨菜单，确认已选择有效音轨。
5. 尝试其他播放源或其他视频。

播放器音量可高于 100。过高增益可能失真，不建议用高增益替代正常系统音量设置。

## 8. 音轨错误或切换失败

### 常见原因

- 文件没有正确标记音轨语言。
- 目标音轨损坏或编码不受当前环境支持。
- 切换时仍在缓冲。
- 切换播放源后音轨列表发生变化。

### 处理步骤

1. 等待媒体加载稳定。
2. 打开音轨菜单重新选择。
3. 观察“音轨切换中”或失败提示。
4. 失败时恢复原音轨。
5. 尝试其他来源。

音轨选择只影响当前播放会话，不修改真实视频文件。

## 9. 字幕菜单为空

### 检查步骤

1. 等待播放器完成媒体和字幕轨发现。
2. 确认视频内确实有嵌入字幕。
3. 确认外挂字幕文件仍存在且可访问。
4. 切换播放源后重新打开字幕菜单。
5. 检查是否选择了“无字幕”。

不同播放源的嵌入字幕不同。切换来源后原音轨或字幕编号可能不再存在。

## 10. 外挂字幕没有自动加载

### 常见原因

- 字幕文件名与视频差异较大。
- 字幕不在可发现位置。
- 字幕文件被移动或删除。
- 字幕格式或编码异常。

### 处理步骤

1. 确认字幕文件可读取。
2. 在字幕菜单中查看外挂字幕候选。
3. 手动选择正确字幕。
4. 如果绑定失效，重新建立绑定。

删除字幕绑定不会删除真实本地字幕源文件。

## 11. 在线字幕搜索失败

### 常见提示和处理

| 提示 | 处理 |
| --- | --- |
| API Key 未配置 | 在设置页填写 OpenSubtitles API Key 并测试 |
| 鉴权失败 | 重新检查 API Key；如配置账号密码，也需重新检查 |
| 无权限或额度受限 | 检查账号权限、下载额度或稍后重试 |
| 服务暂时不可用 | 保留当前播放，稍后重试 |
| 网络失败 | 检查网络、代理和防火墙 |
| 无结果 | 修改标题、类型、语言或关键词 |

只有 API Key 时也可使用界面允许的能力；账号密码是可选项，实际权限由 OpenSubtitles 决定。

## 12. 在线字幕下载失败

### 处理步骤

1. 先在设置页重新测试 OpenSubtitles。
2. 确认搜索结果具有可下载文件标识。
3. 检查网络和剩余磁盘空间。
4. 更换另一个搜索结果。
5. 稍后重试服务端错误。

下载或缓存保存失败时，XFVerse 不应写入无效字幕绑定。

## 13. 在线字幕下载成功但没有显示

### 检查步骤

1. 返回字幕菜单，确认新字幕已选中。
2. 等待字幕切换完成。
3. 检查当前进度是否确实有对白字幕。
4. 尝试前后跳转少量时间。
5. 切换到“无字幕”后再重新选择。
6. 查看是否提示字幕切换较慢或失败。

如果切换来源，需重新确认字幕绑定和当前来源的兼容性。

## 14. 删除在线字幕与清缓存的区别

- 删除绑定：解除字幕与当前电影、集或来源的软件关联。
- 移除临时字幕：停止使用对应临时字幕。
- 清理孤立在线字幕缓存：只删除不再被绑定引用的缓存文件。

上述操作都不会删除视频文件。扫描发现的本地外挂字幕也不属于孤立在线字幕缓存。

## 15. 续播位置不正确

### 常见原因

- 上次播放器异常退出，最后进度尚未保存。
- 切换到时长不同的播放源。
- 原来源不可用。
- 对应记录、历史或进度已删除。
- 视频时长探测不准确。

### 处理步骤

1. 确认当前播放源。
2. 手动调整到正确位置。
3. 正常关闭播放器，使当前进度有机会保存。
4. 再次打开验证。

不要通过强制结束进程测试正常续播。

## 16. 切换播放源失败

### 处理步骤

1. 确认新来源可访问。
2. 等待当前缓冲结束后切换。
3. 失败时返回原来源。
4. 在详情页重新探测新来源。
5. 来源失效时保留其他来源，不要删除整个电影或剧集。

不同版本时长不一致时，切换后的进度可能需要手动调整。

## 17. 上一集或下一集不可用

该功能只适用于 Episode 播放。按钮不可用的常见原因：

- 已经是首集或末集。
- 相邻集没有可用来源。
- 当前内容没有可靠的季集关系。
- 当前播放的是电影。

检查季详情中的集顺序和播放源。

## 18. 播放日志

播放问题优先查看：

```text
%LOCALAPPDATA%\MediaLibrary\Logs\mpv-playback.log
```

视频缓存问题可能还涉及：

```text
%LOCALAPPDATA%\MediaLibrary\Logs\video-cache-debug.log
```

只截取问题时间附近的少量内容，并按[诊断与隐私](诊断与隐私.md)脱敏。

## 仍未解决时提供

- XFVerse 版本和架构。
- 本地或 WebDAV 来源。
- 视频编码、分辨率和大致文件大小。
- 是否有声音、是否有画面。
- 是否只有某一来源失败。
- 错误发生时间和播放位置。
- 脱敏播放日志片段。
- 在线字幕问题需补充错误类别，不提供 API Key。
