## 文档定位

帮助文档按用户问题、错误现象和恢复任务组织，目标是让用户快速定位原因并安全恢复。

它不从头介绍软件，也不复制使用说明书。

## 目标

建立 `docs/help/` 帮助中心，覆盖 1.0 最常见且可验证的问题。

## 建议文档结构

### `docs/help/README.md`

- 按症状导航。
- 快速链接。
- 如何获取版本和诊断信息。
- 如何安全提交问题。

### `docs/help/安装与启动.md`

- 安装器打不开。
- SmartScreen / 未知发布者。
- 不支持的架构。
- 首次启动失败。
- 数据库 migration 失败。
- 覆盖升级后无法启动。
- 修复和完全重装。

### `docs/help/扫描与识别.md`

- 本地目录不可访问。
- WebDAV 测试失败。
- 扫描没有结果。
- 文件被忽略。
- 识别错误或未识别。
- TV 季集无法分组。
- 如何安全复扫和修正。

### `docs/help/播放与字幕.md`

- 播放器打不开或黑屏。
- 远程播放卡顿。
- Range 请求问题。
- 没有声音或音轨不对。
- 字幕未自动加载。
- 在线字幕搜索、登录或下载失败。
- 续播或播放进度异常。

### `docs/help/影片发现与外部服务.md`

- TMDB、OMDb、OpenSubtitles、AI 配置错误。
- 网络、代理、防火墙、DNS 和 TLS 问题。
- API 额度、认证和模型不兼容。
- 哪些功能会降级，哪些仍可使用。

### `docs/help/媒体库与数据操作.md`

- 搜索或筛选无结果。
- 批量操作部分失败。
- 移出媒体库与恢复。
- 删除记录的影响。
- 已移出内容无法恢复。
- 状态冲突或更新失败。

### `docs/help/观影历史收藏与洞察.md`

- 历史为空或无法打开。
- 收藏状态不一致。
- 电影状态互斥。
- 电视剧状态层级。
- 统计为空或失败。
- 画像未配置、数据不足或使用缓存。

### `docs/help/缓存备份与恢复.md`

- 数据目录和备份。
- 海报、元数据、字幕、视频缓存。
- 安全清缓存。
- 数据库副本恢复。
- 卸载后恢复。
- 不删除真实媒体文件的说明。

### `docs/help/诊断与隐私.md`

- 获取应用版本。
- 获取 Windows 版本和架构。
- 查找日志。
- 复制错误信息。
- 脱敏路径、URL、账号和媒体名。
- 禁止提交 API Key、Token、Password 或完整数据库。

## 每个问题的统一模板

1. 现象。
2. 影响范围。
3. 常见原因。
4. 安全检查步骤。
5. 修复步骤。
6. 不要做什么。
7. 仍未解决时需要提供的脱敏信息。
8. 相关使用说明书章节。

## 内容规则

- 不把“删除数据库”作为首选修复。
- 需要重建数据时，先备份并明确影响。
- 不要求用户关闭防火墙或安全软件作为长期方案。
- 不要求用户上传完整数据库或配置文件。
- 不展示完整 WebDAV URL、用户名或路径。
- 命令必须适用于支持的 Windows / PowerShell 环境。
- 所有错误名和日志位置必须来自 RC。
