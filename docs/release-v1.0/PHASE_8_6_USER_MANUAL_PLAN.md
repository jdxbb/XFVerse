## 文档定位

软件使用说明书负责完整、系统、按功能和流程解释“软件怎么用”和“操作会产生什么结果”。

它不是安装说明，也不是按错误症状组织的帮助中心。

## 建议目录

### 第 1 章：认识 XFVerse

- 产品定位。
- 支持的媒体来源。
- 核心概念：媒体记录、播放源、Movie、TV Series、Season、Episode、字幕绑定、用户状态。

### 第 2 章：界面和导航

- 主窗口、侧边栏、标题区、主题切换和用户菜单。

### 第 3 章：首次使用推荐流程

1. 完成个人资料和主题选择。
2. 配置本地目录或 WebDAV。
3. 配置可选元数据和字幕服务。
4. 扫描。
5. 检查识别结果。
6. 打开详情并播放。
7. 使用收藏、历史和洞察。

## 截图和证据规则

- 截图来自最终 RC。
- 使用专门的脱敏演示数据。
- 每个重要页面至少有一张全局图；复杂交互按需要增加局部图。
- 截图中的按钮文案、页面结构和状态必须与说明文字一致。
- 不用历史 DesignDraft 截图冒充成品。

## 关键语义

- 删除记录不删除本地或 WebDAV 文件。
- 移出媒体库只隐藏，保留用户状态和 metadata。
- 清理缓存不删除影片、收藏、进度或真实媒体。
- 删除在线字幕绑定与清理孤立缓存是不同操作。
- TV 不进入 1.0 的观影洞察和 AI 推荐输入。
- AI 返回只作为受控能力，不保证替代 TMDB 或用户确认。
- 外部服务未配置不等于整个软件不可用。
