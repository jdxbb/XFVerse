# XFVerse 1.0 RC 环境与验收矩阵

Last updated: 2026-06-20

## 1. 目的

本文定义 XFVerse 1.0 RC/GA 必须覆盖的环境组合。Phase 8.1 只冻结矩阵，不执行正式 RC 安装验收。

## 2. 环境分级

| 级别 | 含义 |
| --- | --- |
| P0 | GA 前必须通过；失败即阻断发布。 |
| P1 | 高风险核心场景；原则上必须通过，无法覆盖时需明确缩减声明。 |
| P2 | 扩展或兼容性场景；失败可评估 Deferred，但必须记录。 |

## 9. RC 执行记录格式

每次 RC 验收至少记录：

```text
RC 版本：
安装包 SHA-256：
操作系统与构建号：
架构：
显示缩放与主题：
全新安装或升级：
用户数据基线：
媒体来源类型：
外部服务配置状态：
执行的矩阵 ID：
通过：
失败：
Deferred：
日志是否脱敏：
验收人和日期：
```

不得把真实路径、URL、账号、Token、API Key 或私有媒体名写入长期阶段文档。

## 10. GA 通过条件

- 所有 P0 项通过。
- P1 失败项已修复，或明确缩减产品支持声明并更新全部文档。
- P2 项允许 Deferred，但必须写入发布说明。
- 安装包内容审计、数据保留、删除语义、版本一致性、第三方声明和 SHA-256 不允许豁免。
