## 阶段基线

- 本轮只覆盖初级 UI 重构，不做最终视觉。
- 不新增功能，不新增页面，不引入新 UI 框架。
- 不改推荐算法、播放器、扫描识别核心、媒体库删除语义或观影洞察统计口径。
- 不新增数据库字段，不新增 migration。
- UI 初级重构已完成，可进入后续主线阶段。

## UI-1 范围

- 主导航改为：首页、媒体库、影片发现、观影历史、收藏夹、观影洞察。
- 详情页、扫描任务、设置、AI 推荐保留为隐藏路由。
- 删除重复资源页面及 ViewModel、DataTemplate、DI、导航 key。
- 新增影片发现最小页面壳，预留影片搜索、榜单、AI 推荐三个 Tab。
- 新增观影历史最小页面壳，预留日期分组区域。
- 新增个人资料弹窗空壳。
- 导航底部新增假用户区、用户菜单，并把扫描任务和设置入口移入用户菜单。
- 新增导航栏展开 / 收起运行时状态。

## UI-2 范围

- 首页改为 70% / 30% 两栏结构。
- 左侧包含片库预览、继续观看、最近新增。
- 片库预览显示已看、喜爱、想看、不想看四状态总览，不显示未看。
- 继续观看复用现有最近观看数据源，并提供观影历史入口。
- 最近新增复用现有首页入库时间倒序数据源。
- 右侧 AI 推荐预览复用现有 `RecommendationsViewModel`，不复制推荐算法。
- 发现更多影片跳转影片发现页面。
- 本阶段不改推荐服务、不改播放器、不新增数据库字段或 migration。

## UI-3 范围

- 媒体库顶部筛选区重排为搜索 / 清除筛选 + 排序、顺序、范围、标签、年代、收藏状态、识别状态、观看状态。
- 范围筛选采用二级菜单结构，包含影片范围和影片来源。
- 影片范围接入现有列表筛选：全部、库内、库外；库外仅包含库外已看 / 不想看，不包含库外想看。
- 影片来源保留全部来源 / 本地 / WebDAV 入口，第一版为占位，不做真实来源筛选。
- 标签筛选采用固定二维多选菜单，类型标签、情绪标签、场景标签使用固定词表；多个已选标签之间按全部满足过滤。
- 年代筛选改为十年一档的选项，不再使用自由年份输入。
- 识别状态文案调整为自动匹配、待人工确认、手动确认、识别失败、未识别。
- 收藏状态筛选接入喜爱、想看、不想看现有字段，不新增库外喜爱字段。
- 列表上方状态栏显示筛选结果数量，右侧提供布局切换占位按钮和批量选择入口。
- 批量操作、详情跳转、AI 辅助识别、移出媒体库和删除影片记录语义保持不变。
- 本阶段不新增数据库字段或 migration，不做最终卡片视觉。

## UI-4 范围

- 影片发现页继续保留影片搜索、榜单、AI 推荐三个 Tab。
- 影片搜索和榜单仍为占位内容，不接真实功能。
- AI 推荐 Tab 复用现有 `RecommendationsViewModel` 和 `RecommendationsPage` DataTemplate。
- `Recommendations` 隐藏路由继续保留，已有入口仍可打开原推荐页。
- 首页接入影片发现已在 UI-2 完成。
- 本阶段不改推荐算法、不改推荐服务、不新增数据库字段或 migration。

## UI-5 范围

- 观影历史页接入真实 `WatchHistory` 只读查询，按观看日期分组展示。
- 观影历史页按播放记录流水展示，不使用 60 秒有效观看阈值；只展示能关联到 Movie 的历史。
- 同一电影同一天多次观看只显示最新一条；不同日期保留多次。
- 日期筛选支持全部、今天、本周、本月、指定日期。
- 观影洞察日历点击日期跳转观影历史，并以指定日期筛选当天。
- 历史记录展示海报、标题、观看时间、观看时长和播放进度；点击进入详情页。
- 本阶段不改播放器、不改自动已看算法、不改 `WatchHistory` 写入，不新增数据库字段或 migration。

## UI-6 范围

- 收藏夹改为喜爱 / 想看两个 Tab，不再使用全部 / 喜爱 / 想看的下拉筛选作为主结构。
- 喜爱 Tab 显示库内 `Movie.IsFavorite` 影片；当前模型没有库外喜爱字段，因此不伪造库外喜爱。
- 想看 Tab 显示 `UserMovieCollectionItem.IsWantToWatch` 影片，不区分库内 / 库外。
- 不想看不作为收藏夹 Tab 展示，仍保留在推荐硬过滤和媒体库筛选中。
- 同一 TMDB ID 按单条展示，优先使用库内影片信息。
- 取消喜爱走 `IMovieManagementService.SetFavoriteAsync`，取消想看走 `IUserCollectionService.RemoveWantToWatchAsync`，保留状态历史写入。
- 本阶段不新增数据库字段或 migration，不改推荐算法、不改详情页业务逻辑。

## UI-7 范围

- 扫描任务页整理为 70% / 30% 结构：左侧扫描配置和扫描进度，右侧扫描记录。
- 扫描配置左侧接入 WebDAV 扫描路径管理：添加、编辑、删除、启用 / 停用。
- 扫描配置右侧接入 WebDAV URL、用户名、密码、保存设置和测试连接。
- 扫描进度使用现有 `IMediaScanService.RunScanAsync`，扫描中显示 indeterminate 进度，不做假进度。
- 当前扫描服务不支持暂停，不显示假暂停；取消扫描使用已有 cancellation token。
- 扫描记录复用 `ScanTaskLogs` 最近记录，按时间倒序显示扫描目标、用户名、状态、开始 / 结束时间和数量汇总。
- 设置页旧 WebDAV / 扫描路径 UI 已在 UI-8 收口移除。
- 本阶段只做 WebDAV，不新增本地文件夹扫描，不改扫描识别核心逻辑，不新增数据库字段或 migration。

## UI-8 范围

- 设置页整理为两个 Tab：通用、API 配置。
- 通用 Tab 包含缓存设置、行为设置和关于卡片。
- 缓存设置复用现有视频缓存容量、占用刷新、保存设置和清空缓存逻辑。
- 行为设置复用现有主题切换逻辑。
- 关于卡片显示 XFVerse、版本号和占位展开按钮。
- API 配置 Tab 拆分为 TMDB 配置、OMDb 配置和大模型配置。
- TMDB / OMDb 各自拥有独立保存设置、测试连接和状态提示。
- WebDAV 配置和扫描路径管理已从设置页移除，主入口转为扫描任务页。
- 本阶段不新增设置项、不改 API 调用逻辑、不新增数据库字段或 migration。
