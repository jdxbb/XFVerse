# UI 初级重构计划

## 阶段基线

- UI-0 只读审计已完成。
- UI-1 聚焦导航 Shell、重复资源页面删除、最小页面壳和用户区入口。
- 本轮不做最终视觉，不改推荐、扫描、播放器、设置和观影洞察业务逻辑。
- 本轮不新增数据库字段，不新增 migration。

## UI-1 范围

- 主导航改为：首页、媒体库、影片发现、观影历史、收藏夹、观影洞察。
- 详情页、扫描任务、设置、AI 推荐保留为隐藏路由。
- 删除重复资源页面及 ViewModel、DataTemplate、DI、导航 key。
- 新增影片发现最小页面壳，预留影片搜索、榜单、AI 推荐三个 Tab。
- 新增观影历史最小页面壳，预留日期分组区域。
- 新增个人资料弹窗空壳。
- 导航底部新增假用户区、用户菜单，并把扫描任务和设置入口移入用户菜单。
- 新增导航栏展开 / 收起运行时状态。

## UI-2 范围

- 首页改为 70% / 30% 两栏结构。
- 左侧包含片库预览、继续观看、最近新增。
- 片库预览显示已看、喜爱、想看、不想看四状态总览，不显示未看。
- 继续观看复用现有最近观看数据源，并提供观影历史入口。
- 最近新增复用现有首页入库时间倒序数据源。
- 右侧 AI 推荐预览复用现有 `RecommendationsViewModel`，不复制推荐算法。
- 发现更多影片跳转影片发现页面；首页接入指定 AI 推荐 Tab 留到后续增强。
- 本阶段不改推荐服务、不改播放器、不新增数据库字段或 migration。

## UI-4 范围

- 影片发现页继续保留影片搜索、榜单、AI 推荐三个 Tab。
- 影片搜索和榜单仍为占位内容，不接真实功能。
- AI 推荐 Tab 复用现有 `RecommendationsViewModel` 和 `RecommendationsPage` DataTemplate。
- `Recommendations` 隐藏路由继续保留，已有入口仍可打开原推荐页。
- 首页接入影片发现已在 UI-2 完成，直接定位 AI 推荐 Tab 留到后续增强。
- 本阶段不改推荐算法、不改推荐服务、不新增数据库字段或 migration。
