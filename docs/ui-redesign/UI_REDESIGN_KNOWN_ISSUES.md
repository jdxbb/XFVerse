# UI 初级重构 Known Issues

## Blocker

- 无当前 blocker。

## Deferred

- 影片发现的影片搜索 Tab 仍为占位，尚未接入真实搜索。
- 影片发现的榜单 Tab 仍为占位，尚未接入真实榜单。
- 媒体库影片来源筛选保留“全部来源 / 本地 / WebDAV”入口，但当前来源模型只有 WebDAV，真实来源筛选待后续接入。
- 媒体库布局切换按钮为占位，暂不切换真实海报 / 列表布局。
- 本地文件夹扫描支持待后续主线接入。
- 海报缓存待后续主线接入。
- 软件缓存管理待后续主线接入。
- 电视剧支持待后续主线接入。
- 在线字幕搜索待后续主线接入。
- 最终 UI 视觉统一、细节动效和响应式收口待最终 UI 阶段处理。
- 扫描服务暂无逐文件实时进度回调，扫描中使用 indeterminate 进度条；实时进度待服务层支持后接入。
- 扫描服务暂无暂停能力，本阶段不显示假暂停。

## Known Limitation

- 个人资料弹窗使用假头像和假用户名，尚未接入真实用户系统。
- 退出登录为占位按钮，尚未接入登录状态。
- 当前数据模型没有库外喜爱字段，收藏夹喜爱 Tab 只能完整覆盖库内 `Movie.IsFavorite` 影片。
- 库外影片详情仍复用现有未入库详情入口，不新增详情页。
- 收藏夹暂不提供复杂筛选、排序和搜索。
- 观影历史第一版加载最近 100 条去重后的播放记录，尚未做分页或加载更多。
- 观影历史自定义日期使用基础 `DatePicker`，暂未做日期区间选择。
- 观影历史进度依赖 `LastPlayPositionSeconds` 和媒体文件时长 / 影片片长；缺少总时长时显示观看时长与位置文本。
- 从观影洞察日历跳转观影历史后会筛选到指定日期，暂未做列表滚动定位。
- 设置页行为设置当前仅复用已有主题切换，更多行为设置待后续产品阶段接入。
- 设置页关于卡片的展开按钮为占位行为，完整关于详情留到最终 UI 阶段。
- 大模型配置保持既有能力：当前只提供保存配置，未发现既有测试连接命令。

## Noise

- `Recommendations` 隐藏路由继续保留；短期内存在影片发现 AI 推荐 Tab 和隐藏推荐路由两个可达入口。
- 首页“发现更多影片”跳转影片发现默认 Tab，不强制定位到 AI 推荐 Tab。
- 首页 AI 推荐预览使用简化卡片，完整想看 / 不想看操作仍在影片发现 AI 推荐 Tab。
- 最近新增按现有 Movie 维度展示，暂未额外处理同 TMDB 多 Movie 记录的跨记录去重。
- 扫描进度区已识别 / 未识别实时计数暂无服务字段，当前显示 `--`。
- 取消扫描依赖现有 cancellation token；当前 `ScanTaskStatus` 没有 Canceled 状态，取消日志状态后续可单独增强。
