# 播放器页面 UI 实施说明

## 1. 模块名称

播放器

建议文件名：

- `player-page.md`

建议实现对象：

- `PlayerWindow`
- `PlayerView`
- 或现有播放器对应 XAML 文件

---

## 2. 页面用途

播放器用于播放影片播放源。

播放器基于：

- WPF + .NET
- WPF 内嵌 mpv

播放器负责展示视频画面，并提供播放控制、进度控制、缓存进度、音量控制、亮度控制、播放源切换、字幕切换、音轨切换和全屏控制。

播放器可能以弹窗形式出现，也可能进入全屏状态。

---

本模块截图包含 2 张：

1. `01-播放器-弹窗/全屏`

## 4. 播放器整体结构

播放器主要包含：

1. 顶部影片信息区
2. mpv 播放区域
3. 加载状态
4. 左侧亮度浮层
5. 右侧音量浮层
6. 音量悬浮窗
7. 底部控制栏
8. 播放源 / 字幕 / 音轨悬浮菜单

整体结构：

```text
播放器
├─ 顶部影片信息区
│  ├─ 返回按钮 / 关闭播放器
│  └─ 电影名
│
├─ 播放区域
│  ├─ mpv 视频画面
│  ├─ 加载状态
│  ├─ 左侧亮度浮层
│  ├─ 右侧音量浮层
│  └─ 音量悬浮窗
│
└─ 底部控制栏
   ├─ 播放 / 暂停按钮
   ├─ 当前播放时间
   ├─ 播放进度条
   ├─ 缓存进度条
   ├─ 总时长
   ├─ 音量按钮
   ├─ 音量百分比
   ├─ 播放源按钮
   ├─ 字幕按钮
   ├─ 音轨按钮
   └─ 全屏按钮
```

---

## 5. 顶部影片信息区

顶部左侧包含：

- 返回按钮
- 电影名

### 返回按钮

左上角黑色方块是返回按钮，也就是关闭播放器按钮。

点击后：

- 关闭播放器
- 返回上一页面或上一状态
- 具体关闭逻辑按现有播放器逻辑处理

要求：

- 使用 WPF UI 返回图标或关闭图标
- 不使用 App Logo
- 深色播放器背景下保持清晰可见

### 电影名

显示当前播放影片名。

要求：

- 电影名过长时省略
- 鼠标悬浮可显示完整电影名 Tooltip
- 不遮挡视频主体内容

---

## 6. 播放区域

播放区域为 mpv 视频承载区域。

要求：

- 视频画面优先显示
- 未加载或无画面时使用 WPF / XAML 深色沉浸式背景
- 不准备播放器背景图片
- 不使用截图中的橙色背景
- 不使用真实电影画面作为 UI 背景
- 不使用强霓虹或赛博朋克风格

---

## 7. 加载状态

播放区域中央显示加载状态。

内容包括：

- 加载图标
- 加载百分比

示例：

```text
加载中 xx%
```

要求：

- 加载图标使用 WPF UI 图标 / 旋转进度图标 / ProgressRing
- 加载百分比来自当前加载或缓冲状态
- 加载状态居中显示
- 加载完成后隐藏
- 不阻挡底部控制栏操作，除非现有逻辑要求

---

## 8. 底部控制栏

底部控制栏浮在播放器底部。

显示 / 隐藏逻辑：

- 按现有播放器逻辑处理
- 不在本 UI spec 中重新定义

底部控制栏包含：

- 播放 / 暂停按钮
- 当前播放时间
- 播放进度条
- 缓存进度条
- 总时长
- 音量按钮
- 音量百分比
- 播放源按钮
- 字幕按钮
- 音轨按钮
- 全屏按钮

要求：

- 底部控制栏使用半透明深灰 / 黑灰浮层
- 控件保持可读
- 不遮挡太多视频内容
- 控件间距清晰
- 支持弹窗和全屏状态

---

## 9. 播放 / 暂停按钮

底部功能栏最左边的黑色正方形是播放 / 暂停按钮。

行为：

- 播放中显示暂停图标
- 暂停中显示播放图标
- 点击后在播放和暂停之间切换

说明：

- 播放 / 暂停切换按现有播放器逻辑处理
- 不重新定义播放状态逻辑

---

## 10. 播放进度和缓存进度

底部进度条需要同时表达：

- 当前播放进度
- 缓存进度

显示内容：

- 当前播放时间
- 播放进度条
- 缓存进度条
- 总时长

要求：

- 播放进度使用主色灰粉 / 浅粉
- 缓存进度使用半透明浅灰或中性色
- 进度条可拖动逻辑按现有播放器逻辑处理
- 时间格式按现有逻辑处理，例如 `hh:mm:ss`
- 当前进度和总时长保持对齐

---

## 11. 音量规则

音量范围：

- `0% - 200%`

其中：

- `0% - 100%` 为普通音量
- `100% - 200%` 为音量增强

要求：

- 普通音量部分使用普通音量颜色
- 超过 100% 的增强部分单独变色
- 超出 100% 的部分使用低饱和橙色
- 超过 150% 时可加深橙色
- 不使用刺眼颜色
- 音量百分比显示当前音量值

底部音量文字示例：

```text
音量：xx
```

或：

```text
音量：xxx%
```

具体格式按现有播放器逻辑处理。

---

## 12. 音量按钮

底部音量图标按钮用于静音。

行为：

- 单击音量图标直接静音
- 如果已有取消静音逻辑，按现有播放器逻辑处理
- 鼠标悬停到音量区域时，显示音量悬浮窗

说明：

- 单击音量图标不是打开菜单
- 音量悬浮窗由悬停音量区域触发

---

## 13. 音量悬浮窗

截图中间偏下的竖向条是鼠标悬停到音量区域时出现的音量悬浮窗。

内容包括：

- 音量图标
- 竖向音量条
- 当前音量百分比

规则：

- 音量范围 0% - 200%
- 0% - 100% 为普通音量色
- 100% - 200% 为音量增强色
- 超过 100% 的部分变色
- 悬浮窗在鼠标离开音量区域后按现有逻辑隐藏

要求：

- 音量悬浮窗不遮挡关键播放内容
- 浮层使用半透明深灰 / 黑灰视觉
- 深色播放器背景下可读
- 交互按现有逻辑接入

---

## 14. 左侧亮度浮层

左侧竖向浮层是亮度浮层。

亮度范围：

- `0% - 100%`

内容包括：

- 亮度图标
- 竖向亮度条
- 百分比

要求：

- 亮度条只显示 0% - 100%
- 亮度条颜色与普通音量颜色一致
- 亮度条不使用音量增强颜色
- 浮层出现 / 隐藏逻辑按现有播放器逻辑处理
- 百分比显示当前亮度值

---

## 15. 右侧音量浮层

右侧竖向浮层是音量浮层。

音量范围：

- `0% - 200%`

内容包括：

- 音量图标
- 竖向音量条
- 百分比

要求：

- 0% - 100% 使用普通音量颜色
- 100% - 200% 使用音量增强颜色
- 超过 100% 的部分变色
- 浮层出现 / 隐藏逻辑按现有播放器逻辑处理
- 百分比显示当前音量值

---

## 16. 播放源 / 字幕 / 音轨按钮

底部控制栏包含：

- 播放源按钮
- 字幕按钮
- 音轨按钮

行为：

- 单击播放源按钮，出现播放源悬浮菜单
- 单击字幕按钮，出现字幕悬浮菜单
- 单击音轨按钮，出现音轨悬浮菜单
- 悬停这些按钮本身没有特殊变化，除通用 hover 反馈外
- 不通过悬停按钮打开菜单

---

## 17. 字幕菜单

字幕菜单通过点击字幕按钮打开。

菜单内容包括：

- 无
- 内嵌
- 外挂

结构示例：

```text
字幕菜单
├─ 无
├─ 内嵌 》
└─ 外挂 》
```

### 二级菜单

悬停在 `内嵌` 或 `外挂` 后出现二级菜单。

二级菜单显示对应字幕列表。

示例：

```text
简体中文
```

### 当前字幕状态

当前字幕项需要高亮。

要求：

- 当前字幕项使用高亮背景或主色强调
- 不要求显示勾选图标
- 字幕菜单可以滚动
- 字幕数量较多时，菜单内部滚动
- 滚动条使用统一样式

---

## 18. 音轨菜单

音轨菜单通过点击音轨按钮打开。

菜单内容为可用音轨列表。

示例：

```text
音轨 1
音轨 2
```

### 当前音轨状态

当前音轨项需要高亮。

要求：

- 当前音轨项使用高亮背景或主色强调
- 不要求显示勾选图标
- 音轨菜单可以滚动
- 音轨数量较多时，菜单内部滚动
- 滚动条使用统一样式

---

## 19. 播放源菜单

播放源菜单通过点击播放源按钮打开。

菜单为表格结构。

表头：

- 播放源
- 分辨率
- 码率
- 大小

内容示例：

```text
播放源 1    分辨率 1    码率 1    大小 1
播放源 2    分辨率 2    码率 2    大小 2
播放源 3    分辨率 3    码率 3    大小 3
```

行为：

- 点击播放源菜单项后切换播放源
- 切换播放源后的具体播放逻辑按现有播放器逻辑处理

要求：

- 当前播放源可高亮，若现有逻辑支持
- 播放源菜单可以滚动
- 播放源数量较多时，菜单内部滚动
- 表头保持清晰
- 滚动条使用统一样式

---

## 20. 全屏按钮

底部控制栏右侧有全屏按钮。

行为：

- 全屏逻辑按现有播放器逻辑处理
- 图标变化按现有逻辑处理

要求：

- 全屏按钮始终清晰可见
- 全屏状态下不破坏底部控制栏布局
- 弹窗和全屏状态下都应可操作

---

## 21. 可滚动区域

播放器主界面无页面滚动区域。

菜单允许内部滚动：

- 字幕菜单可以滚动
- 音轨菜单可以滚动
- 播放源菜单可以滚动

要求：

- 播放器页面整体不滚动
- mpv 播放区域不滚动
- 底部控制栏不滚动
- 菜单内容过多时只在菜单内部滚动

---

## 22. 状态处理

### 未加载

未加载时：

- 显示深色沉浸式背景
- 显示加载状态或空状态
- 不使用独立播放器背景图片

### 加载中

加载中：

- 中央显示加载图标和百分比
- 底部控制栏可按现有逻辑显示或隐藏

### 播放中

播放中：

- 播放 / 暂停按钮显示暂停图标
- 进度条更新
- 缓存进度更新

### 暂停中

暂停中：

- 播放 / 暂停按钮显示播放图标

### 静音

静音时：

- 音量图标应反映静音状态
- 再次点击是否恢复音量按现有逻辑处理

---

## 23. 深色 / 浅色主题注意事项

播放器区域永远使用深色沉浸式风格。

要求：

- 播放器不跟随普通页面浅色主题变成浅色播放器
- 播放器背景使用深灰 / 黑灰
- 控制栏使用半透明深灰 / 黑灰
- 菜单使用半透明深灰浮层
- 播放进度条使用灰粉主色
- 缓存进度使用半透明浅灰
- 普通音量和亮度使用浅粉 / 灰粉主色
- 音量增强使用低饱和橙色
- 不使用强霓虹
- 不使用赛博朋克风格
- 不使用强烈发光效果

---

- 播放中显示暂停图标
- 暂停中显示播放图标
- 显示当前播放时间

- 显示音量百分比
- 显示播放源按钮
- 显示字幕按钮
- 显示音轨按钮
- 显示全屏按钮

### 音量和亮度验收

- 单击音量图标可以静音
- 鼠标悬停音量区域显示音量悬浮窗
- 音量范围支持 0% - 200%
- 0% - 100% 显示普通音量颜色
- 100% - 200% 显示音量增强颜色
- 超过 100% 的部分单独变色
- 左侧亮度浮层显示亮度
- 亮度范围支持 0% - 100%
- 亮度条不使用音量增强颜色
- 右侧音量浮层显示音量
- 右侧音量浮层支持 0% - 200%

### 菜单验收

- 单击播放源按钮打开播放源菜单
- 单击字幕按钮打开字幕菜单
- 单击音轨按钮打开音轨菜单
- 悬停播放源 / 字幕 / 音轨按钮本身不打开菜单
- 字幕菜单包含无、内嵌、外挂
- 悬停内嵌显示内嵌字幕二级菜单
- 悬停外挂显示外挂字幕二级菜单
- 当前字幕项高亮
- 音轨菜单显示可用音轨
- 当前音轨项高亮
- 播放源菜单显示播放源、分辨率、码率、大小
- 字幕菜单内容多时可滚动
- 音轨菜单内容多时可滚动
- 播放源菜单内容多时可滚动
- 点击播放源菜单项后的逻辑按现有播放器逻辑处理

### 全屏验收

- 全屏按钮可按现有逻辑进入或退出全屏
- 全屏状态下控制栏仍可正常显示和隐藏
- 全屏状态下菜单和浮层位置正常
