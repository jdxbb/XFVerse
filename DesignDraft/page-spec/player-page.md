# 播放器页面 UI 实施说明

## 1. 模块名称

播放器

建议文件名：

- `player-page.md`

建议实现对象：

- `PlayerWindow`
- `PlayerView`
- 或现有播放器对应 XAML 文件

---

## 2. 页面用途

播放器用于播放影片播放源。

播放器基于：

- WPF + .NET
- WPF 内嵌 mpv

播放器负责展示视频画面，并提供播放控制、进度控制、可用时的 mpv 缓冲反馈、音量控制、亮度控制、播放源切换、字幕切换、在线字幕搜索与下载、音轨切换和全屏控制。

播放器可能以弹窗形式出现，也可能进入全屏状态。

## 3.1 Phase 6.0f-B 确认后的最终设计基线

本节覆盖本文中与其冲突的旧实现假设。

- 播放器沿用旧设计的深色沉浸式布局、顶部返回 / 关闭浮层、底部控制栏、音量 / 亮度反馈和菜单层级。
- 播放器窗口不使用原生系统标题栏，使用设计化的最小化、最大化 / 还原、关闭按钮。
- 在线字幕搜索弹窗属于播放器链路，同样固定使用深色视觉，不随主应用浅色 / 深色主题切换。
- 控制栏补充电视剧播放所需的 `上一集` / `下一集`；当前成品中的 `停止` 不作为最终设计组件。
- 字幕菜单最终包含 `无字幕`、`内嵌`、`外挂`、`在线下载字幕` 四类。
- 在线字幕搜索窗口、绑定展示、下载 / 错误 / 额度状态详见 `online-subtitle-search-page.md`。
- `播放进度` 与 `缓冲进度` 是两种不同语义；当前不把离线缓存作为已实现的播放器能力，也不伪造离线缓存进度。
- 字幕选择和在线字幕入口只属于播放器链路；电影、剧、季、集详情页不新增字幕入口。

## 4. 播放器整体结构

播放器主要包含：

1. 自定义窗口控制与顶部影片信息区
2. mpv 播放区域
3. loading / buffering / error / notice 状态层
4. 左侧亮度浮层
5. 右侧音量浮层
6. 音量悬浮窗
7. 底部控制栏
8. 播放源 / 字幕 / 音轨悬浮菜单
9. 在线字幕搜索深色弹窗

整体结构：

```text
播放器
├─ 自定义窗口控制与顶部影片信息区
│  ├─ 返回按钮 / 关闭播放器
│  ├─ 当前播放标题
│  └─ 最小化 / 最大化或还原 / 关闭窗口
│
├─ 播放区域
│  ├─ mpv 视频画面
│  ├─ loading / buffering / error / notice 状态层
│  ├─ 左侧亮度浮层
│  ├─ 右侧音量浮层
│  └─ 音量悬浮窗
│
└─ 底部控制栏
   ├─ 播放 / 暂停按钮
   ├─ 上一集 / 下一集（剧集会话）
   ├─ 当前播放时间
   ├─ 播放进度条
   ├─ 可选缓冲进度视觉（仅 mpv 提供可靠进度时）
   ├─ 总时长
   ├─ 音量按钮
   ├─ 音量百分比
   ├─ 播放源按钮
   ├─ 字幕按钮
   ├─ 音轨按钮
   └─ 全屏按钮
```

---

## 5. 顶部影片信息区

顶部浮层包含：

- 返回按钮
- 当前播放标题
- 自定义窗口按钮：最小化、最大化 / 还原、关闭

最终播放器窗口不保留原生系统窗口标题栏。自定义窗口按钮应与播放器深色浮层共用视觉基线，并在弹窗与全屏切换状态下保持可发现、可点击。

`返回 / 关闭播放器` 承载退出当前播放会话的入口语义；自定义窗口关闭按钮承载窗口控制。若当前实现中两者最终均关闭播放窗口，Phase 7 应复用同一安全关闭行为，不新增另一套业务动作。

### 返回按钮

左上角黑色方块是返回按钮，也就是关闭播放器按钮。

点击后：

- 关闭播放器
- 返回上一页面或上一状态
- 具体关闭逻辑按现有播放器逻辑处理

要求：

- 使用 WPF UI 返回图标或关闭图标
- 不使用 App Logo
- 深色播放器背景下保持清晰可见

### 当前播放标题

显示当前播放电影名，或剧集会话中的剧名、季集编号与集名摘要。

要求：

- 标题过长时省略
- 鼠标悬浮可显示完整标题 Tooltip
- 不遮挡视频主体内容

---

## 6. 播放区域

播放区域为 mpv 视频承载区域。

要求：

- 视频画面优先显示
- 未加载或无画面时使用 WPF / XAML 深色沉浸式背景
- 不准备播放器背景图片
- 不使用截图中的橙色背景
- 不使用真实电影画面作为 UI 背景
- 不使用强霓虹或赛博朋克风格

---

## 7. 播放状态浮层

播放区域中央显示 opening、loading metadata、starting、seeking、buffering、recovering、error 或 notice 状态。

内容可包括：

- 加载或缓冲图标
- 状态文案
- 可可靠获取时的进度百分比
- 错误后的关闭或重试类既有操作

要求：

- 加载图标使用 WPF UI 图标 / 旋转进度图标 / ProgressRing
- 只有 mpv 能提供可靠百分比时才显示缓冲数值；否则仅显示 `加载中` 或 `缓冲中` 文案和反馈动画
- 不把 `buffering` 状态包装成离线缓存下载进度
- 普通状态浮层居中显示，状态结束后隐藏；错误与 notice 的关闭行为应明确
- 不阻挡底部控制栏操作，除非当前播放状态无法接受操作

---

## 8. 底部控制栏

底部控制栏浮在播放器底部。

显示 / 隐藏逻辑：

- 按现有播放器逻辑处理
- 不在本 UI spec 中重新定义

底部控制栏包含：

- 播放 / 暂停按钮
- 上一集 / 下一集按钮，仅在剧集上下文适用时可用
- 当前播放时间
- 播放进度条
- 可选缓冲进度视觉，仅在 mpv 可可靠提供时显示
- 总时长
- 音量按钮
- 音量百分比
- 播放源按钮
- 字幕按钮
- 音轨按钮
- 全屏按钮

要求：

- 底部控制栏使用半透明深灰 / 黑灰浮层
- 控件保持可读
- 不遮挡太多视频内容
- 控件间距清晰
- 支持弹窗和全屏状态
- `停止` 按钮不作为最终控制栏组件

## 9. 播放 / 暂停按钮

底部功能栏最左边的黑色正方形是播放 / 暂停按钮。

说明：

- 不重新定义播放状态逻辑

## 10. 播放进度、缓冲进度和离线缓存边界

| 概念 | 最终设计语义 | 显示规则 |
|---|---|---|
| 播放进度 | 当前播放位置 | 使用主色灰粉 / 浅粉，支持现有 seek 行为 |
| 缓冲进度 | mpv 对当前播放会话的预加载 / 缓冲反馈 | 仅在可可靠获得百分比时以中性辅助轨显示 |
| `loading` / `buffering` | 无可靠百分比时的会话等待状态 | 显示状态浮层，不伪造条形进度 |
| 离线缓存 | 当前项目没有真正播放器离线缓存能力 | 不在播放器最终设计中展示离线缓存进度 |

显示内容：

- 当前播放时间
- 播放进度条
- 可选缓冲反馈
- 总时长

要求：

- 播放进度与缓冲反馈不能混用文案或颜色含义
- 缓冲反馈只能反映本次会话可获得的 mpv 缓冲信息
- 进度条可拖动逻辑按现有播放器逻辑处理
- 时间格式按现有逻辑处理，例如 `hh:mm:ss`
- 当前进度和总时长保持对齐

## 11. 音量规则

音量范围：

- `0% - 200%`

其中：

- `0% - 100%` 为普通音量
- `100% - 200%` 为音量增强

要求：

- 普通音量部分使用普通音量颜色
- 超过 100% 的增强部分单独变色
- 超出 100% 的部分使用低饱和橙色
- 超过 150% 时可加深橙色
- 不使用刺眼颜色
- 音量百分比显示当前音量值

底部音量文字示例：

```text
音量：xx
```

或：

```text
音量：xxx%
```

具体格式按现有播放器逻辑处理。

## 17. 字幕菜单

字幕菜单通过点击字幕按钮打开。

最终菜单分类包括：

- 无字幕
- 内嵌
- 外挂
- 在线下载字幕

结构示例：

```text
字幕菜单
├─ 无字幕
├─ 内嵌 》
├─ 外挂 》
└─ 在线下载字幕 》
```

### 二级菜单

悬停在 `内嵌`、`外挂` 或 `在线下载字幕` 后出现二级菜单。

二级菜单显示对应字幕列表。

`在线下载字幕` 二级菜单包括：

- 搜索在线字幕入口
- 已下载 / 已绑定在线字幕列表
- 当前选中标记
- 缓存缺失或绑定失效提示
- 每条绑定旁的删除绑定入口

打开搜索窗口时，如果当前正在播放则先暂停；关闭搜索窗口后不自动恢复播放。用户回到播放器后自行决定是否继续播放。

### 当前字幕状态

当前字幕项需要高亮。

### 在线字幕绑定展示语义

以下只定义 UI 展示语义，不扩展后端绑定或识别逻辑：

| 当前播放对象 | 菜单归属文案 |
|---|---|
| 已识别电影 | 绑定到当前电影 |
| 已识别剧集单集 | 绑定到当前集 |
| 未识别播放项 | 绑定到当前播放文件 |

该归属文案用于说明删除绑定影响范围、缓存缺失 / 绑定失效提示，以及识别修正后的显示关系。

### 删除绑定确认

- 删除绑定入口旁显示小型 Popover / 小弹窗确认，不使用全局居中大弹窗
- 文案必须说明：仅删除绑定，不删除字幕缓存文件
- 物理清理字幕缓存归软件缓存管理，不在播放器菜单中执行

要求：

- 当前字幕项使用高亮背景或主色强调
- 不要求显示勾选图标
- 字幕菜单可以滚动
- 字幕数量较多时，菜单内部滚动
- 滚动条使用统一样式
- 下载成功后，新字幕加入在线字幕列表并自动切换为当前字幕
- 关闭字幕、切换失败、字幕加载失败均需要有菜单内或播放器状态提示
- 在线字幕搜索弹窗的完整规格见 `online-subtitle-search-page.md`

## 19. 播放源菜单

播放源菜单通过点击播放源按钮打开。

菜单采用旧设计的菜单行结构：顶部为类似表头的信息行，下方为可选择的播放源菜单行。该结构不是传统表格，也不是只显示标题的简略菜单。

表头：

- 播放源
- 分辨率
- 码率
- 大小

内容示例：

```text
播放源 1    分辨率 1    码率 1    大小 1
播放源 2    分辨率 2    码率 2    大小 2
播放源 3    分辨率 3    码率 3    大小 3
```

行为：

- 点击播放源菜单项后切换播放源
- 切换播放源后的具体播放逻辑按现有播放器逻辑处理
- 结合当前需要保留的按钮，例如切换、默认源标记或必要状态操作；不把所有探测字段堆入菜单

要求：

- 当前播放源可高亮，若现有逻辑支持
- 播放源菜单可以滚动
- 播放源数量较多时，菜单内部滚动
- 表头保持清晰
- 滚动条使用统一样式
- 探测内容标签仅以旧设计表头字段为基线：播放源 / 分辨率 / 码率 / 大小；其它必要状态优先用徽标或操作提示承载
- 文件路径可以展示；完整 WebDAV URL 不得裸露，应转为安全路径摘要或来源文案
- 文件名、路径、release 名称等长字段超出宽度时省略，鼠标悬停显示允许展示的完整内容
- 需要设计无播放源、播放源不可用、WebDAV 失效和切换失败状态

## 22. 状态处理

播放器状态矩阵：

| 状态 | 视觉反馈 | 交互说明 |
|---|---|---|
| `opening` | 深色背景与打开中提示 | 播放控件可暂时 disabled |
| `loading metadata` | 居中 loading 与读取媒体信息文案 | 不显示伪造进度 |
| `starting` | 启动播放提示 | 等待可播放首帧 |
| `seeking` | 轻量跳转中反馈 | 保持当前 seek 语义 |
| `buffering` | 缓冲提示；可靠时可显示缓冲百分比 | 不等同于离线缓存 |
| `recovering` | 恢复播放提示 | 表示会话恢复，不误导为重新下载 |
| `error` | 错误 overlay 与可执行既有操作 | 不遮掩错误原因 |
| `notice` | 短时操作反馈 overlay | 用于切换、音量、字幕等反馈 |
| 播放中 | 播放 / 暂停按钮显示暂停图标，播放进度更新 | 播放源与菜单可操作 |
| 暂停中 | 播放 / 暂停按钮显示播放图标 | 保持当前画面 |
| 静音 | 音量图标反映静音状态 | 再次点击恢复逻辑按现有能力 |

未加载或无可播放源时使用深色空态 / 错误态，不使用独立播放器背景图片。

## 23. 深色 / 浅色主题注意事项

播放器区域永远使用深色沉浸式风格。

要求：

- 播放器不跟随普通页面浅色主题变成浅色播放器
- 播放器背景使用深灰 / 黑灰
- 控制栏使用半透明深灰 / 黑灰
- 菜单使用半透明深灰浮层
- 播放进度条使用灰粉主色
- 可靠的缓冲进度可使用半透明浅灰；无可靠进度时只显示 buffering 状态
- 普通音量和亮度使用浅粉 / 灰粉主色
- 音量增强使用低饱和橙色
- 不使用强霓虹
- 不使用赛博朋克风格
- 不使用强烈发光效果
- 在线字幕搜索弹窗及播放器内 Menu / Popup / Popover 固定使用播放器深色基线

## 28. 验收标准

该播放器完成后，需要满足：

### 基础验收

- 播放器可以承载 mpv 播放画面
- 播放器可作为弹窗或全屏显示
- 播放器使用深色沉浸式风格
- 播放器不受普通页面浅色主题影响
- 播放器与在线字幕搜索窗口固定深色，播放器使用自定义现代窗口按钮
- 左上角显示返回 / 关闭播放器按钮
- 点击左上角按钮关闭播放器
- 顶部显示当前播放标题
- 当前播放标题过长时省略并可查看完整内容
- 项目可以编译通过

### 加载验收

- 加载中显示居中加载图标
- 仅在可可靠获得数值时显示加载或缓冲百分比
- 加载完成后隐藏加载状态
- opening、loading metadata、starting、seeking、buffering、recovering、error、notice 均有对应视觉状态

### 底部控制栏验收

- 底部控制栏显示 / 隐藏按现有逻辑
- 底部最左侧显示播放 / 暂停按钮
- 剧集上下文显示上一集 / 下一集入口
- 不显示停止按钮作为最终控件
- 播放中显示暂停图标
- 暂停中显示播放图标
- 显示当前播放时间
- 显示总时长
- 显示播放进度
- 有可靠 mpv 缓冲进度时可显示缓冲反馈；不得显示伪造的离线缓存进度
- 显示音量按钮
- 显示音量百分比
- 显示播放源按钮
- 显示字幕按钮
- 显示音轨按钮
- 显示全屏按钮

### 音量和亮度验收

- 单击音量图标可以静音
- 鼠标悬停音量区域显示音量悬浮窗
- 音量范围支持 0% - 200%
- 0% - 100% 显示普通音量颜色
- 100% - 200% 显示音量增强颜色
- 超过 100% 的部分单独变色
- 左侧亮度浮层显示亮度
- 亮度范围支持 0% - 100%
- 亮度条不使用音量增强颜色
- 右侧音量浮层显示音量
- 右侧音量浮层支持 0% - 200%

### 菜单验收

- 单击播放源按钮打开播放源菜单
- 单击字幕按钮打开字幕菜单
- 单击音轨按钮打开音轨菜单
- 悬停播放源 / 字幕 / 音轨按钮本身不打开菜单
- 字幕菜单包含无字幕、内嵌、外挂、在线下载字幕
- 悬停内嵌显示内嵌字幕二级菜单
- 悬停外挂显示外挂字幕二级菜单
- 悬停在线下载字幕显示搜索入口和已绑定在线字幕列表
- 当前字幕项高亮
- 音轨菜单显示可用音轨
- 当前音轨项高亮
- 播放源菜单显示播放源、分辨率、码率、大小
- 播放源菜单采用表头式菜单行结构，文件路径遵守脱敏与长字段规则
- 删除在线字幕绑定通过小型确认 Popover 完成，并明确不删除缓存文件
- 字幕菜单内容多时可滚动
- 音轨菜单内容多时可滚动
- 播放源菜单内容多时可滚动
- 点击播放源菜单项后的逻辑按现有播放器逻辑处理

### 全屏验收

- 全屏按钮可按现有逻辑进入或退出全屏
- 全屏状态下控制栏仍可正常显示和隐藏
- 全屏状态下菜单和浮层位置正常
