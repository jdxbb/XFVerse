扫描任务

建议实现对象：

- `ScanTasksPage`

---

## 2. 页面用途

扫描任务页面用于管理 WebDAV 与本地目录两类扫描来源，分别启动扫描任务，并在共享区域查看运行进度和历史日志。

该页面是由用户菜单进入的全屏功能页，不是弹窗。本规格已按 Phase 6.0g-B 用户确认结果更新；与旧截图或旧文字说明冲突时，以当前规则为准。

---

## 3. 设计来源与最终调整

参考截图：

- `screenshots/账号/扫描任务/01-扫描任务-全屏.png`

保留旧设计：

- 全屏任务页框架
- 配置 / 进度 / 历史记录的信息层级
- 扫描历史卡片结构
- 任务状态的紧凑可读表达

补齐和调整：

- 页面覆盖 WebDAV 与本地目录两个明确分区，不合并成含混路径输入。
- WebDAV 与本地扫描分别启动，共享进度区和历史日志区。
- 最终设计不提供暂停按钮，只保留开始与取消等当前实际能力。
- 扫描识别计数为条件性结果摘要，不长期展示无真实值的占位状态。
- 历史日志采用旧设计卡片结构，但按安全规则隐藏敏感内容。
- 删除本地扫描路径使用按钮旁轻量确认 Popover。

## 4. 页面整体结构

```text
扫描任务页面
├─ 顶部标题区
│  ├─ 返回按钮
│  ├─ 标题：扫描任务
│  └─ 副标题：管理 WebDAV 与本地目录扫描任务
├─ 配置区
│  ├─ WebDAV 扫描分区
│  │  ├─ 连接设置
│  │  └─ WebDAV 路径列表

├─ 共享扫描进度区
```

布局规则：

- 保留旧设计清晰区分配置、进度和历史的结构。
- 页面可按实际内容纵向滚动；路径列表与日志列表较长时优先在各自区域内滚动。
- WebDAV 和本地目录必须以独立标题和独立操作呈现。

## 5. WebDAV 扫描分区

### 5.1 连接设置

字段与操作：

| 内容 | 设计规则 |
| --- | --- |
| WebDAV URL / endpoint | 配置输入可编辑；日志和历史记录不得裸露完整 URL |
| 用户名 | 配置表单可编辑；日志区域使用安全摘要，不展示完整账号 |
| 密码 | 使用敏感字段眼睛图标显示 / 隐藏规则 |
| 启用连接 | 开关或复选框 |
| 保存 | 只保存，不自动测试 |
| 测试连接 | 用户主动执行 |

状态：

- 未配置
- 保存中 / 保存成功 / 保存失败
- 测试中 / 连接成功 / 鉴权失败 / 网络错误 / 路径不可访问
- disabled，例如没有启用连接或未填写必要信息

### 5.2 WebDAV 路径列表

支持：

- 添加路径
- 编辑显示名或安全路径配置
- 启用 / 禁用
- 递归扫描开关
- 移除路径

语义：

- 扫描只读远端媒体来源。
- 扫描页不得提供删除 WebDAV 远端真实文件的能力。
- 路径与 URL 的展示遵循长字段及脱敏规则。

---

## 6. 本地目录扫描分区

本地目录扫描作为与 WebDAV 并列的正式能力。

每个本地目录项支持：

- 路径选择或输入
- 显示名称
- 启用 / 禁用
- 递归设置
- 编辑
- 删除路径配置
- 启动本地扫描

状态：

- 暂无目录
- 路径不可访问
- 路径不存在
- 权限不足
- 空目录
- disabled
- 保存成功 / 保存失败

安全语义：

- 本地扫描不删除真实本地文件。
- 删除路径配置可能影响对应软件记录的可见性，但不删除真实本地文件。
- 删除本地路径配置必须使用按钮旁确认 Popover，详见第 10 节及 `global-dialogs.md`。

---

## 7. 扫描启动与运行控制

最终操作区包含：

| 按钮 | 语义 |
| --- | --- |
| `开始扫描网盘` | 启动 WebDAV 扫描任务 |
| `开始扫描本地文件` | 启动本地目录扫描任务 |
| `取消扫描` | 取消当前运行中的任务 |

规则：

- WebDAV 和本地扫描分别启动，但共享进度和日志区域。
- 扫描运行中，不可重复触发的按钮显示 disabled。
- 不保留 `暂停扫描` 按钮，不把未存在的暂停能力设计为最终 UI。
- 取消扫描只取消软件任务，不删除来源文件。

## 8. 扫描进度与统计

### 8.1 实时运行状态

扫描中可以展示：

- 当前来源：WebDAV 或本地目录
- 当前阶段
- 当前正在处理的文件或安全路径摘要
- 已扫描数量
- 总数或估算总数，仅在可可靠获取时展示
- 新增数量
- 更新数量
- 跳过数量
- 错误数量
- 已耗时

长字段规则：

- 当前文件或路径过长时单行省略。
- 悬停仅显示允许展示的完整内容。
- 完整 WebDAV URL、凭据或敏感远端路径信息不得裸露。

### 8.2 条件性的识别计数

`已识别` / `未识别` 属于扫描完成摘要中的条件性指标：

- 如果扫描流程能够稳定完成识别并返回可靠统计，则在扫描结束后展示。
- 不要求扫描执行过程中实时更新识别计数。
- 如果流程不能稳定返回该统计，则最终设计不显示这两个指标。
- 不显示长期为 `--` 的伪完成占位。

### 8.3 完成状态

完成摘要需区分：

- 成功
- 部分成功
- 失败
- 已取消

每种结果均可显示对应统计和安全错误摘要，但不显示敏感路径或凭据。

## 9. 扫描历史与日志

扫描历史保留旧设计的紧凑任务卡结构，WebDAV 和本地任务共享同一个历史日志区域。

任务卡可以包含：

- 安全来源标签：`WebDAV` 或 `本地目录`
- 安全目标摘要或用户设置的显示名称
- 任务状态
- 开始 / 结束时间
- 已扫描、新增、更新、跳过、错误数量
- 耗时
- 原因摘要或最主要失败原因
- 条件可用时的识别结果摘要

显示安全规则：

- WebDAV 记录只显示安全目标摘要或别名，不显示完整 URL。
- 账号信息不得作为历史卡片明文内容。
- token、password、API key 永不进入日志。
- 本地路径和其它允许展示的长字段可省略，悬停显示完整允许内容。
- 不支持的数据字段不得用固定占位符伪装为有效统计。

状态类型：

- pending
- running
- success
- partial success
- failed
- cancelled
- empty

---

## 10. 删除本地扫描路径轻量确认

删除本地扫描路径属于局部高影响操作，最终设计采用删除按钮旁的小 Popover，而不是默认使用全局居中大弹窗。

建议内容：

```text
标题：移除此扫描路径？
正文：将删除该扫描路径配置，并可能影响对应软件记录的可见性。不会删除真实本地文件。
取消按钮：取消
确认按钮：移除
```

交互规则：

- Popover 锚定在当前路径行的删除按钮旁。
- 用户必须显式确认或取消。
- 点击外部或按 Esc 等同取消。
- 确认按钮采用警示语义，不写成删除真实文件。
- 此操作不新增真实文件删除能力。

---

## 11. 敏感字段、路径和日志规则

- WebDAV 密码采用眼睛图标显示 / 隐藏控件，默认隐藏。
- 其它 token 或 API key 若在扫描关联设置中出现，同样采用敏感字段规则。
- 保存只执行保存，不自动触发连接测试。
- 测试连接由独立按钮执行。
- WebDAV URL 不在历史或日志区域裸露。
- 日志文本不得包含密码、token 或 API key。
- 所有长字段遵循 `codex-ui-rules.md` 的省略与 Tooltip 规则。

## 13. Phase 7 实施注意事项

- Phase 7 只按已具备的数据和命令呈现 UI；不因视觉规格私自增加扫描业务能力。
- 已识别 / 未识别指标需在数据能力确认后才落入实现。
- 不新增暂停扫描。
- 不泄露远端地址、账号或密钥。
- 不改变扫描只读来源文件的安全语义。

---

## 14. 验收标准

- 页面呈现 WebDAV 与本地目录两个清晰扫描分区。
- WebDAV 与本地任务分别启动，并共享进度与历史日志区域。
- 最终设计包含开始和取消操作，不包含暂停扫描按钮。
- WebDAV 密码和其它敏感字段采用眼睛图标显示 / 隐藏方案。
- 保存与测试连接是独立动作。
- 页面覆盖配置缺失、测试中、连接失败、路径不可访问、扫描中、部分成功、失败、取消和 disabled 状态。
- 扫描中的实时统计不伪造识别结果。
- 识别计数仅在流程稳定产出时于扫描结束后显示，否则不纳入最终设计。
- 扫描历史保持旧设计卡片结构，并增加本地来源、原因摘要和安全脱敏规则。
- WebDAV URL、账号与凭据不在日志中裸露。
- 删除本地扫描路径使用按钮旁小 Popover，并明确不删除真实本地文件。
