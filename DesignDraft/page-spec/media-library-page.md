# 媒体库页面 UI 实施说明

## 1. 模块名称

媒体库

建议文件名：

- `media-library-page.md`

建议实现对象：

- `MediaLibraryPage`
- 或 `MediaLibraryPage.xaml`

---

## 2. 页面用途

媒体库页面用于浏览、搜索、筛选、排序、扫描和批量管理影片资源。

用户从左侧导航栏点击“媒体库”进入该页面。

页面支持：

- 展开导航栏状态
- 关闭导航栏状态
- 海报布局
- 列表布局
- 关闭导航栏状态下的批量选择模式

页面不是弹窗，是完整页面。

## 16. 状态标签显示规则

海报卡片右上角的状态标签，以及列表项右侧的状态标签，显示影片当前最重要状态。

状态规则：

- 影片基础观看状态为：已看 / 未看
- 已看影片才能标记喜爱
- 未看影片才能标记想看
- 已看和未看影片都可以标记不想看
- 不想看 / 想看 / 喜爱 三个状态互斥，任意两个都不能共存
- 不想看会覆盖想看和喜爱
- 不想看优先级最高

### 互斥规则

- 如果影片标记为不想看，则不能同时是想看或喜爱
- 如果影片原本是想看，再标记不想看，则取消想看，显示不想看
- 如果影片原本是喜爱，再标记不想看，则取消喜爱，显示不想看
- 如果影片原本是不想看，再标记想看或喜爱，需要按现有后端逻辑先取消不想看，再设置新状态
- 三个偏好状态中，同一时间最多只能存在一个：不想看 / 想看 / 喜爱

### 已看影片

已看影片状态优先级：

1. 不想看
2. 喜爱
3. 已看

规则：

- 如果已看影片标记了不想看，显示“不想看”
- 否则如果已看影片标记了喜爱，显示“喜爱”
- 否则显示“已看”

### 未看影片

未看影片状态优先级：

1. 不想看
2. 想看
3. 未看

规则：

- 如果未看影片标记了不想看，显示“不想看”
- 否则如果未看影片标记了想看，显示“想看”
- 否则显示“未看”

### 视觉要求

状态标签颜色遵循 `DESIGN.md`：

- 已看：蓝色
- 未看：淡灰边框，不填充
- 想看：浅粉色
- 喜爱：比想看更深的浅粉
- 不想看：低饱和橙红，不刺眼

## 17. 批量选择模式

批量选择模式只出现在关闭导航栏状态。

展开导航栏状态下不显示批量选择按钮。

### 进入批量选择

关闭导航栏状态下显示按钮：

- `批量选择`

点击后：

- 进入批量选择模式
- 按钮变为 `完成`
- 显示批量操作按钮组
- 海报卡片或列表项出现选择控件

### 退出批量选择

按钮文案：

- `完成`

点击后：

- 退出批量选择模式
- 隐藏批量操作按钮组
- 清空或保留选中项按现有逻辑处理
- 按钮恢复为 `批量选择`

### 批量选择模式下点击行为

批量选择模式下：

- 点击海报卡片切换选中
- 点击列表项切换选中
- 不进入详情页

### 选择控件位置

海报布局批量选择：

- 每张海报卡片下方显示圆形选择控件

列表布局批量选择：

- 每条列表项右侧显示圆形选择控件

选择控件应：

- 有选中 / 未选中状态
- 选中态使用主色或低饱和高亮
- 未选中态使用中性色边框
- 不遮挡影片主要信息

## 18. 批量操作按钮组

批量选择模式下显示批量操作按钮组。

按钮包括：

- `AI 辅助识别`
- `标记为已看`
- `标记为未看`
- `删除记录`

这些按钮只对已选中的影片生效。

如果没有选中任何影片：

- 批量操作按钮禁用
- 不需要点击后再提示
- 禁用态应清晰可见

### AI 辅助识别

作用：

- 对已选影片执行 AI 辅助识别

具体业务逻辑按现有后端处理。

### 标记为已看

作用：

- 将已选影片标记为已看

具体状态规则按现有后端处理。

### 标记为未看

作用：

- 将已选影片标记为未看

具体状态规则按现有后端处理。

### 删除记录

作用：

- 删除已选影片记录

要求：

- 点击后必须弹出二次确认弹窗
- 用户确认后才执行删除
- 用户取消则不删除
- 删除成功后刷新媒体库列表和结果摘要
- 删除失败时显示错误提示

二次确认弹窗建议文案：

- 标题：`确认删除记录？`
- 说明：`删除后将从媒体库记录中移除所选影片。`
- 确认按钮：`删除`
- 取消按钮：`取消`

## 22. 空状态

如果没有媒体库数据，使用 EmptyState 组件。

可能文案：

- 标题：`暂无影片`
- 说明：`扫描媒体库后，这里会显示你的影片资源。`

如果搜索 / 筛选后没有结果，使用 EmptyState 组件。

可能文案：

- 标题：`没有找到影片`
- 说明：`换个关键词或清除筛选条件试试。`

要求：

- 不使用独立插画图片
- 使用 WPF UI 图标 + 文案 + 可选按钮
- 空状态只出现在媒体库列表区

该页面完成后，需要满足：

### 基础验收

- 从左侧导航栏点击“媒体库”可以进入媒体库页面
- 当前导航项高亮为“媒体库”
- 页面显示标题“媒体库”
- 页面显示副标题“查看你的影片资源”
- 页面右上角显示扫描媒体库按钮
- 页面右上角显示主题切换按钮
- 页面支持浅色 / 深色主题
- 项目可以编译通过

### 扫描媒体库验收

- 点击“扫描媒体库”触发扫描任务开始扫描
- 点击后跳转到扫描任务页面

### 搜索筛选验收

- 搜索框显示“搜索影片、导演、演员”
- 点击搜索按钮后触发搜索
- 筛选工具区包含范围、标签、年代、收藏状态、识别状态、观看状态
- 点击选项按钮弹出对应下拉菜单
- 清除筛选按钮可以清空筛选条件
- 结果摘要显示共找到 xx 部影片
- 筛选后结果摘要更新

### 布局验收

- 媒体库支持海报布局
- 媒体库支持列表布局
- 点击切布局可以在海报布局和列表布局之间切换
- 下次进入媒体库时保持上次选择的布局
- 媒体库列表区可滚动
- 顶部标题区不滚动
- 搜索筛选区不滚动
- 结果摘要区不滚动

### 海报布局验收

- 海报布局显示影片网格
- 每张海报卡片显示来源标签
- 每张海报卡片显示状态标签
- 每张海报卡片显示海报
- 每张海报卡片显示电影名
- 每张海报卡片显示日期
- 每张海报卡片显示类别标签和情绪标签
- 每张海报卡片显示观看进度条和百分比
- 普通模式下点击海报进入详情页
- 海报缺失时使用 poster-placeholder.png

### 列表布局验收

- 列表布局显示横向长条列表项
- 列表项显示平均评分
- 列表项显示电影名和原名
- 列表项显示日期和时长
- 列表项显示导演和演员
- 列表项显示类别标签、情绪标签和观影场景
- 列表项显示观看进度条和百分比
- 列表项显示来源标签
- 列表项显示状态标签
- 普通模式下点击列表项进入详情页

### 状态标签验收

- 已看影片无其他状态时显示“已看”
- 未看影片无其他状态时显示“未看”
- 已看影片标记喜爱时显示“喜爱”
- 未看影片标记想看时显示“想看”
- 已看或未看影片标记不想看时显示“不想看”
- 不想看优先级最高
- 已看组优先级为：不想看 > 喜爱 > 已看
- 未看组优先级为：不想看 > 想看 > 未看
- 不想看、想看、喜爱三个状态互斥，任意两个不能共存
- 不想看会覆盖想看和喜爱
- 影片标记不想看后，状态标签显示“不想看”
- 原本想看的影片标记不想看后，不再显示“想看”
- 原本喜爱的影片标记不想看后，不再显示“喜爱”

### 批量选择验收

- 批量选择只出现在关闭导航栏状态
- 点击批量选择后进入批量选择模式
- 批量选择按钮变为完成
- 批量选择模式下显示 AI 辅助识别、标记为已看、标记为未看、删除记录
- 未选中任何影片时批量操作按钮禁用
- 海报布局批量选择时，海报卡片下方显示选择控件
- 列表布局批量选择时，列表项右侧显示选择控件
- 批量选择模式下点击卡片切换选中，不进入详情页
- 批量选择模式下点击列表项切换选中，不进入详情页
- 点击完成后退出批量选择模式
- 点击删除记录时弹出二次确认弹窗
- 确认后才删除记录
- 取消后不删除记录
