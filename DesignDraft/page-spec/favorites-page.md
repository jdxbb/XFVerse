# 收藏夹页面 UI 实施说明

## 1. 模块名称

收藏夹

建议文件名：

- `favorites-page.md`

建议实现对象：

- `FavoritesPage`
- 或 `FavoritesPage.xaml`

---

## 2. 页面用途

收藏夹页面用于展示用户收藏相关的影片，包括：

- 喜爱影片
- 想看影片

用户从左侧导航栏点击“收藏夹”进入该页面。

该页面是完整页面，不是弹窗。

---

## 3. 截图说明

本模块截图包含 2 张：

1. `01-收藏夹-展开导航栏`
2. `02-收藏夹-关闭导航栏`

截图中的颜色不参与最终 UI 设计。

截图颜色只用于区分区域和信息结构。  
最终视觉必须遵循：

- `DESIGN.md`
- `resources-note.md`
- `codex-ui-rules.md`

实际 UI 应使用：

- WPF UI
- 深色 / 浅色双主题
- 灰粉 / 中性色视觉体系
- 低饱和语义色
- WPF UI 默认图标优先

---

## 4. 页面整体结构

收藏夹页面主要包含：

1. 左侧导航栏
2. 顶部标题区
3. Tab 区
4. 影片列表区域

整体结构：

```text
收藏夹页面
├─ 左侧导航栏
│  ├─ 展开状态
│  └─ 关闭状态
│
├─ 顶部标题区
│  ├─ 标题：收藏夹
│  ├─ 副标题：收藏你的独家喜好
│  └─ 主题切换按钮
│
├─ Tab 区
│  ├─ 喜爱
│  └─ 想看
│
└─ 影片列表区域
   └─ 影片卡片网格
```

---

## 5. 顶部标题区

顶部标题区包含：

- 页面标题
- 页面副标题
- 主题切换按钮

### 页面标题

标题文本：

- `收藏夹`

### 页面副标题

副标题文本：

- `收藏你的独家喜好`

说明：

- 展开导航栏状态下，截图中主要显示标题
- 关闭导航栏状态下，标题旁显示副标题
- 实际实现可根据布局空间显示副标题
- 不应因为副标题导致标题区拥挤

### 主题切换按钮

页面右上角显示主题切换按钮。

要求：

- 使用 WPF UI 图标按钮样式
- 图标可根据当前主题显示日间 / 夜间图标
- 不应抢占页面主视觉

---

## 6. 导航栏状态

收藏夹页面支持：

- 展开导航栏状态
- 关闭导航栏状态

### 展开导航栏状态

展开状态下：

- 左侧显示完整导航栏
- 当前选中菜单为“收藏夹”
- 主内容区位于导航栏右侧
- 影片列表区域显示在 Tab 下方

### 关闭导航栏状态

关闭状态下：

- 左侧导航栏隐藏或收缩
- 顶部左侧显示导航栏展开按钮
- 页面标题显示为“收藏夹”
- 副标题显示为“收藏你的独家喜好”
- 主内容区向左扩展

说明：

- 关闭导航栏状态不支持批量选择
- 不显示批量选择按钮
- 不显示批量操作栏

---

## 7. Tab 区

收藏夹页面包含两个 Tab：

- `喜爱`
- `想看`

默认选中：

- `喜爱`

要求：

- 每次进入收藏夹页面默认打开“喜爱”Tab
- Tab 只显示名称，不显示数量
- 当前选中的 Tab 需要有明显高亮
- 选中态使用主色灰粉 / 浅粉点缀
- 可使用下划线或其他轻量选中标记
- 切换 Tab 时刷新影片列表区域
- 不需要记住上次选中的 Tab

---

## 8. 喜爱 Tab

喜爱 Tab 展示用户标记为喜爱的影片。

说明：

- 喜爱状态只适用于已看影片
- 页面只展示当前仍处于喜爱状态的影片
- 如果用户在详情页取消喜爱，该影片应从喜爱 Tab 中移除

页面本身不提供取消喜爱的按钮。

---

## 9. 想看 Tab

想看 Tab 展示用户标记为想看的影片。

说明：

- 想看状态只适用于未看影片
- 页面只展示当前仍处于想看状态的影片
- 如果用户在详情页取消想看，该影片应从想看 Tab 中移除

页面本身不提供取消想看的按钮。

---

## 10. 影片列表区域

影片列表区域是本页面主要内容区。

内容使用海报卡片网格布局。

要求：

- 复用媒体库海报卡片的视觉风格
- 但不显示状态标签
- 右上角显示评分
- 影片列表区域内部滚动
- 顶部标题区和 Tab 区不滚动

---

## 11. 影片卡片

每张影片卡片包含：

- 海报
- 右上角评分
- 电影名
- 日期
- 类别标签
- 情绪标签

结构示例：

```text
影片卡片
├─ 海报
├─ 右上角评分
├─ 电影名
├─ 日期 yyyy-mm-dd
├─ 类别标签 | 情绪标签
```

### 右上角评分

截图中海报右上角的蓝色长方形表示评分，不是状态标签。

要求：

- 显示影片评分
- 不显示“喜爱”“想看”“已看”“未看”“不想看”等状态
- 评分显示格式按现有数据逻辑处理
- 评分缺失时可留空或显示轻量占位，按现有逻辑处理

### 电影名

电影名应清晰可读。

要求：

- 过长时最多显示两行或按现有影片卡片规则处理
- 避免因电影名长短导致卡片高度明显跳动

### 海报缺失

如果影片没有海报：

- 使用 `poster-placeholder.png`

---

## 12. 点击行为

收藏夹中的影片卡片可以点击。

点击行为：

- 点击整张影片卡片进入影片详情页

要求：

- 不需要单独按钮
- 不需要继续播放按钮
- 不需要取消收藏按钮
- 不需要批量选择
- 点击卡片只进入影片详情页

---

## 13. 不支持的操作

收藏夹页面不需要以下功能：

- 不需要搜索
- 不需要筛选
- 不需要排序
- 不需要批量选择
- 不需要批量操作栏
- 不需要取消喜爱按钮
- 不需要取消想看按钮
- 不需要删除按钮
- 不需要扫描媒体库按钮

如果后续新增功能，需要单独补充规格，不由 Codex 自行添加。

---

## 14. 可滚动区域

收藏夹页面的可滚动区域是：

- 影片列表区域

要求：

- 顶部标题区不滚动
- Tab 区不滚动
- 左侧导航栏不滚动
- 影片列表区域内部滚动
- 滚动条使用 WPF UI / 项目统一样式
- 不要给每个 Tab 内容单独添加额外复杂滚动层级
- 切换 Tab 后，影片列表可回到顶部，具体按现有逻辑处理

---

## 15. 空状态

如果喜爱 Tab 没有影片，使用 EmptyState 组件。

建议文案：

- 标题：`暂无喜爱影片`
- 说明：`在影片详情页标记喜爱后，这里会展示你的心头好。`

如果想看 Tab 没有影片，使用 EmptyState 组件。

建议文案：

- 标题：`暂无想看影片`
- 说明：`在影片详情页标记想看后，这里会展示你准备观看的影片。`

要求：

- 空状态显示在影片列表区域
- 不使用独立插画图片
- 使用 WPF UI 图标 + 文案 + 可选操作按钮
- 不影响顶部标题区和 Tab 区

---

## 16. 深色 / 浅色主题注意事项

页面视觉遵循 `DESIGN.md`。

### 浅色主题

- 页面背景为淡粉白或统一浅色背景
- 影片卡片使用浅色或清晰表面
- Tab 选中态使用主色灰粉 / 浅粉
- 评分标签使用低饱和点缀色
- 滚动条使用统一样式

### 深色主题

- 页面背景使用深灰
- 影片卡片使用深灰 / 黑灰表面
- Tab 选中态使用深色主题下可读的主色
- 使用边框和透明度区分卡片层级
- 不使用强烈发光效果
- 不使用纯黑大面积背景，除非是海报占位或播放器区域

---

## 17. 响应式和窗口尺寸

收藏夹页面需要适配：

- 展开导航栏状态
- 关闭导航栏状态
- 不同窗口尺寸

要求：

- 设置合理最小窗口尺寸
- 窗口变小时允许压缩间距
- 影片卡片网格列数可根据可用宽度调整
- 影片列表区域保持滚动
- Tab 区不能被挤压到不可读

---

## 18. 组件拆分建议

建议拆分为：

```text
FavoritesPage
├─ FavoritesHeader
├─ FavoritesTabs
├─ FavoritesMovieList
└─ FavoriteMovieCard
```

### 组件说明

#### FavoritesPage

负责：

- 页面整体布局
- 导航栏展开 / 关闭状态适配
- 当前导航项选中状态
- 默认打开喜爱 Tab

#### FavoritesHeader

负责：

- 页面标题
- 页面副标题
- 主题切换按钮

#### FavoritesTabs

负责：

- 喜爱 / 想看 Tab 切换
- Tab 选中态

#### FavoritesMovieList

负责：

- 当前 Tab 下的影片网格
- 列表滚动
- 空状态

#### FavoriteMovieCard

负责：

- 海报
- 评分
- 电影名
- 日期
- 类别标签
- 情绪标签
- 点击进入详情页

## 22. 验收标准

该页面完成后，需要满足：

### 基础验收

- 从左侧导航栏点击“收藏夹”可以进入收藏夹页面
- 当前导航项高亮为“收藏夹”
- 页面显示标题“收藏夹”
- 页面显示副标题“收藏你的独家喜好”
- 页面右上角显示主题切换按钮
- 页面不显示“扫描媒体库”按钮
- 页面支持浅色 / 深色主题
- 项目可以编译通过

### Tab 验收

- 页面包含“喜爱”和“想看”两个 Tab
- 每次进入页面默认选中“喜爱”
- Tab 只显示名称，不显示数量
- 当前选中的 Tab 有清晰高亮
- 点击“想看”后切换到想看影片列表
- 点击“喜爱”后切换到喜爱影片列表

### 影片列表验收

- 影片列表区域可滚动
- 顶部标题区不滚动
- Tab 区不滚动
- 左侧导航栏不滚动
- 喜爱 Tab 显示喜爱影片
- 想看 Tab 显示想看影片
- 没有影片时显示 EmptyState

### 影片卡片验收

- 影片卡片显示海报
- 影片卡片右上角显示评分
- 影片卡片右上角不显示状态标签
- 影片卡片显示电影名
- 影片卡片显示日期
- 影片卡片显示类别标签和情绪标签
- 海报缺失时使用 poster-placeholder.png
- 点击整张影片卡片进入影片详情页

### 不支持功能验收

- 不显示搜索框
- 不显示筛选控件
- 不显示排序控件
- 不显示批量选择按钮
- 不显示批量操作栏
- 不显示取消喜爱按钮
- 不显示取消想看按钮
- 不显示删除按钮
