# 收藏夹页面 UI 实施说明

## 1. 模块名称

收藏夹

建议文件名：

- `favorites-page.md`

建议实现对象：

- `FavoritesPage`
- 或 `FavoritesPage.xaml`

---

## 2. 页面用途

收藏夹页面用于展示用户收藏相关的 Movie 与 Season，包括：

- 喜爱项
- 想看项

用户从左侧导航栏点击“收藏夹”进入该页面。

该页面是完整页面，不是弹窗。

---

## 3. 截图说明

本模块截图包含 2 张：

1. `01-收藏夹-展开导航栏`
2. `02-收藏夹-关闭导航栏`

截图中的颜色不参与最终 UI 设计。

截图颜色只用于区分区域和信息结构。  
最终视觉必须遵循：

- `DESIGN.md`
- `resources-note.md`
- `codex-ui-rules.md`

实际 UI 应使用：

- WPF UI
- 深色 / 浅色双主题
- 灰粉 / 中性色视觉体系
- 低饱和语义色
- WPF UI 默认图标优先

## 4. 页面整体结构

收藏夹页面主要包含：

1. 左侧导航栏
2. 顶部标题区
3. Tab 区
4. 收藏列表区域

整体结构：

```text
收藏夹页面
├─ 左侧导航栏
│  ├─ 展开状态
│  └─ 关闭状态
│
├─ 顶部标题区
│  ├─ 标题：收藏夹
│  ├─ 副标题：收藏你的独家喜好
│  └─ 主题切换按钮
│
├─ Tab 区
│  ├─ 喜爱
│  └─ 想看
│
└─ 收藏列表区域
   └─ Movie / Season 卡片网格
```

---

## 5. 顶部标题区

顶部标题区包含：

- 页面标题
- 页面副标题
- 主题切换按钮

### 页面标题

标题文本：

- `收藏夹`

### 页面副标题

副标题文本：

- `收藏你的独家喜好`

说明：

- 展开导航栏状态下，截图中主要显示标题
- 关闭导航栏状态下，标题旁显示副标题
- 实际实现可根据布局空间显示副标题
- 不应因为副标题导致标题区拥挤

### 主题切换按钮

页面右上角显示主题切换按钮。

要求：

- 使用 WPF UI 图标按钮样式
- 图标可根据当前主题显示日间 / 夜间图标
- 不应抢占页面主视觉

---

## 6. 导航栏状态

收藏夹页面支持：

- 展开导航栏状态
- 关闭导航栏状态

### 展开导航栏状态

展开状态下：

- 左侧显示完整导航栏
- 当前选中菜单为“收藏夹”
- 主内容区位于导航栏右侧
- 收藏列表区域显示在 Tab 下方

### 关闭导航栏状态

关闭状态下：

- 左侧导航栏隐藏或收缩
- 顶部左侧显示导航栏展开按钮
- 页面标题显示为“收藏夹”
- 副标题显示为“收藏你的独家喜好”
- 主内容区向左扩展

说明：

- 关闭导航栏状态不支持批量选择
- 不显示批量选择按钮
- 不显示批量操作栏

---

## 7. Tab 区

收藏夹页面包含两个 Tab：

- `喜爱`
- `想看`

默认选中：

- `喜爱`

要求：

- 每次进入收藏夹页面默认打开“喜爱”Tab
- Tab 显示名称与当前数量
- 当前选中的 Tab 需要有明显高亮
- 选中态使用主色灰粉 / 浅粉点缀
- 可使用下划线或其他轻量选中标记
- 切换 Tab 时刷新收藏列表区域
- 不需要记住上次选中的 Tab

---

## 8. 喜爱 Tab

喜爱 Tab 展示用户标记为喜爱的 Movie 或 Season。

说明：

- 页面只展示当前仍处于喜爱状态的收藏项。
- 页面内保留取消喜爱动作；成功后该项从当前 Tab 移除并更新数量。
- 取消中需显示 loading / disabled，失败时保留卡片并给出错误反馈。
- 库外或暂无播放源的喜爱项也可以取消，不为该情况设计固定禁用规则。

---

## 9. 想看 Tab

想看 Tab 展示用户标记为想看的 Movie 或 Season。

说明：

- 页面只展示当前仍处于想看状态的收藏项。
- 页面内保留取消想看动作；成功后该项从当前 Tab 移除并更新数量。
- 取消中需显示 loading / disabled，失败时保留卡片并给出错误反馈。

---

## 10. 收藏列表区域

收藏列表区域是本页面主要内容区。

内容使用海报卡片网格布局。

要求：

- 复用旧设计海报卡片的信息风格，并补入当前 Movie / Season 真实状态
- 右上角显示评分
- 无播放源状态和当前收藏状态可以作为轻量标签展示
- 收藏列表区域内部滚动
- 顶部标题区和 Tab 区不滚动

---

## 11. 收藏卡片

每张卡片共用内容包括：

- 海报
- 右上角评分
- 标题
- 类型适用的辅助字段与标签
- 无播放源或可用性状态
- 页内取消当前收藏状态的动作

结构示例：

```text
收藏卡片
├─ 海报
├─ 右上角评分
├─ 标题
├─ 类型辅助信息
├─ 标签 / 来源状态
└─ 取消喜爱或取消想看动作
```

### 右上角评分

截图中海报右上角的蓝色长方形表示评分，不是状态标签。

要求：

- 显示影片评分
- 评分仍是主视觉角标；收藏状态与无播放源状态使用独立、克制的辅助标签
- 评分显示格式按现有数据逻辑处理
- 评分缺失时可留空或显示轻量占位，按现有逻辑处理

### 字段矩阵

| 类型 | 必需字段 | 可选字段 | 详情目标 |
|---|---|---|---|
| Movie | 海报、电影名、评分或空态、收藏状态 | 日期、类别 / 情绪标签、播放源状态 | Movie detail |
| Season | 海报、剧名、季名 / 季号、收藏状态 | 集数、来源摘要、播放源状态、季适用标签 | Season detail |

要求：

- Movie 与 Season 不强制使用完全相同标签。
- 标题过长时最多显示两行或按全局卡片规则处理，避免高度跳动。

### 海报缺失

如果收藏项没有海报：

- 使用 `poster-placeholder.png`

---

## 12. 点击行为

收藏夹中的收藏卡片可以点击。

点击行为：

- Movie 卡片进入电影详情页。
- Season 卡片进入季详情页。
- 无播放源项仍可进入对应详情。

要求：

- 不需要继续播放按钮
- 保留页内取消喜爱 / 取消想看动作，该动作不触发整卡导航
- 不需要批量选择
- 点击卡片非动作区域只进入对应详情页

---

## 13. 不支持的操作

收藏夹页面不需要以下功能：

- 不需要搜索
- 不需要筛选
- 不需要排序
- 不需要批量选择
- 不需要批量操作栏
- 不需要删除按钮
- 不需要扫描媒体库按钮

取消喜爱 / 取消想看是本页保留操作，默认不使用全局大弹窗确认；需要覆盖 loading、成功、失败和 disabled 状态。

如果后续新增功能，需要单独补充规格，不由 Codex 自行添加。

---

## 14. 可滚动区域

收藏夹页面的可滚动区域是：

- 收藏列表区域

要求：

- 顶部标题区不滚动
- Tab 区不滚动
- 左侧导航栏不滚动
- 收藏列表区域内部滚动
- 滚动条使用 WPF UI / 项目统一样式
- 不要给每个 Tab 内容单独添加额外复杂滚动层级
- 切换 Tab 后，收藏列表可回到顶部，具体按现有逻辑处理

---

## 15. 空状态

如果喜爱 Tab 没有影片，使用 EmptyState 组件。

建议文案：

- 标题：`暂无喜爱影片`
- 说明：`在影片详情页标记喜爱后，这里会展示你的心头好。`

如果想看 Tab 没有影片，使用 EmptyState 组件。

建议文案：

- 标题：`暂无想看影片`
- 说明：`在影片详情页标记想看后，这里会展示你准备观看的影片。`

要求：

- 空状态显示在收藏列表区域
- 不使用独立插画图片
- 使用 WPF UI 图标 + 文案 + 可选操作按钮
- 不影响顶部标题区和 Tab 区

---

## 16. 深色 / 浅色主题注意事项

页面视觉遵循 `DESIGN.md`。

### 浅色主题

- 页面背景为淡粉白或统一浅色背景
- 收藏卡片使用浅色或清晰表面
- Tab 选中态使用主色灰粉 / 浅粉
- 评分标签使用低饱和点缀色
- 滚动条使用统一样式

### 深色主题

- 页面背景使用深灰
- 收藏卡片使用深灰 / 黑灰表面
- Tab 选中态使用深色主题下可读的主色
- 使用边框和透明度区分卡片层级
- 不使用强烈发光效果
- 不使用纯黑大面积背景，除非是海报占位或播放器区域

---

## 17. 响应式和窗口尺寸

收藏夹页面需要适配：

- 展开导航栏状态
- 关闭导航栏状态
- 不同窗口尺寸

要求：

- 设置合理最小窗口尺寸
- 窗口变小时允许压缩间距
- 收藏卡片网格列数可根据可用宽度调整
- 收藏列表区域保持滚动
- Tab 区不能被挤压到不可读

## 18. 组件拆分建议

建议拆分为：

```text
FavoritesPage
├─ FavoritesHeader
├─ FavoritesTabs
├─ FavoritesItemList
└─ FavoriteItemCard
```

### 组件说明

#### FavoritesPage

负责：

- 页面整体布局
- 导航栏展开 / 关闭状态适配
- 当前导航项选中状态
- 默认打开喜爱 Tab

#### FavoritesHeader

负责：

- 页面标题
- 页面副标题
- 主题切换按钮

#### FavoritesTabs

负责：

- 喜爱 / 想看 Tab 切换
- Tab 选中态
- 两个 Tab 的数量

#### FavoritesItemList

负责：

- 当前 Tab 下的 Movie / Season 网格
- 列表滚动
- 空状态

#### FavoriteItemCard

负责：

- 海报
- 评分
- Movie / Season 类型字段矩阵
- 标签与来源可用性状态
- 取消喜爱 / 取消想看及反馈状态
- 点击进入对应详情页

## 22. 验收标准

该页面完成后，需要满足：

### 基础验收

- 从左侧导航栏点击“收藏夹”可以进入收藏夹页面
- 当前导航项高亮为“收藏夹”
- 页面显示标题“收藏夹”
- 页面显示副标题“收藏你的独家喜好”
- 页面右上角显示主题切换按钮
- 页面不显示“扫描媒体库”按钮
- 页面支持浅色 / 深色主题
- 项目可以编译通过

### Tab 验收

- 页面包含“喜爱”和“想看”两个 Tab
- 每次进入页面默认选中“喜爱”
- Tab 显示各自数量
- 当前选中的 Tab 有清晰高亮
- 点击“想看”后切换到想看收藏列表
- 点击“喜爱”后切换到喜爱收藏列表

### 收藏列表验收

- 收藏列表区域可滚动
- 顶部标题区不滚动
- Tab 区不滚动
- 左侧导航栏不滚动
- 喜爱 Tab 显示喜爱 Movie / Season
- 想看 Tab 显示想看 Movie / Season
- 没有收藏项时显示 EmptyState

### 收藏卡片验收

- 收藏卡片显示海报和评分或对应空态
- Movie 卡片显示电影适用字段
- Season 卡片显示剧名、季号 / 季名与季适用字段
- Movie / Season 可使用不同标签矩阵
- 无播放源状态可识别，且不阻止详情导航
- 海报缺失时使用 poster-placeholder.png
- Movie 进入电影详情，Season 进入季详情
- 页内可取消喜爱 / 取消想看并覆盖 loading、成功、失败、disabled
- 库外收藏项允许取消，不显示产品规则型禁用

### 不支持功能验收

- 不显示搜索框
- 不显示筛选控件
- 不显示排序控件
- 不显示批量选择按钮
- 不显示批量操作栏
- 不显示删除按钮
