# 在线字幕搜索窗口 UI 实施说明

## 1. 模块名称

在线字幕搜索窗口

建议文件名：

- `online-subtitle-search-page.md`

建议实现对象：

- `OnlineSubtitleSearchWindow`
- `OnlineSubtitleSearchViewModel`
- 播放器字幕菜单中的在线字幕入口

---

## 2. 页面用途与边界

在线字幕搜索窗口属于播放器链路，用于为当前播放会话搜索、下载并切换在线字幕。

最终设计边界：

- 窗口从播放器字幕菜单的 `在线下载字幕` 二级菜单打开。
- 窗口固定使用播放器深色视觉，不随主应用浅色 / 深色主题切换。
- 打开窗口时，如当前正在播放则暂停；关闭窗口后不自动恢复播放。
- 下载成功后将新字幕加入已绑定在线字幕列表，并自动切换为当前字幕。
- OpenSubtitles 配置在设置页维护；物理缓存清理在软件缓存管理中执行。
- 设置页的 OpenSubtitles 凭据遵循敏感字段显示 / 隐藏规则；本窗口不提供凭据编辑。
- 电影、剧、季、集详情页不新增在线字幕入口。
- 不设计全库自动下载、OCR、翻译或字幕编辑器。

---

## 3. 窗口与整体结构

在线字幕搜索窗口与播放器共用深色窗口基线，并使用设计化的标题栏和窗口按钮。

```text
OnlineSubtitleSearchWindow
├─ CustomWindowChrome
│  ├─ Title: 在线字幕搜索
│  └─ Minimize / Maximize or Restore / Close
├─ SearchForm
│  ├─ LanguageSelect
│  ├─ MediaTypeSelect
│  ├─ KeywordInput
│  ├─ SortSelect
│  └─ SearchButton
├─ ContextSummary
├─ StatusNotice
└─ ResultsList
   └─ OnlineSubtitleResultRow
      ├─ SubtitleMetadata
      ├─ MatchSummary
      └─ DownloadButton
```

要求：

- 弹窗内容在窗口内滚动，标题栏与搜索条件区保持可见。
- 结果多时只滚动结果区。
- 深色背景下保证文本、错误、警告与按钮对比度。

---

## 4. 打开、暂停与关闭行为

| 触发条件 | 设计行为 |
|---|---|
| 从播放器打开，当前正在播放 | 打开窗口前暂停播放，并显示当前搜索上下文 |
| 从播放器打开，当前已暂停 | 保持暂停状态 |
| 用户关闭窗口 | 关闭弹窗，不自动恢复播放 |
| 下载并切换成功后关闭 | 仍不自动恢复播放，由用户主动播放 |

这些行为是播放器体验语义，不要求本设计阶段修改播放逻辑。

---

## 5. 搜索条件

搜索表单包含：

- 语言下拉框，默认中文，并提供静态语言列表。
- 类型下拉框：`电影` / `电视剧集`。
- 关键词输入框。
- 排序下拉框：`综合`、`下载量`、`评分`、`上传时间`、`匹配度` 等当前支持项。
- 搜索按钮。

自动填充语义：

| 播放上下文 | 默认填充内容 |
|---|---|
| 已识别电影 | 电影标题、年份等可用于匹配的信息 |
| 已识别单集 | 剧名、季号、集号等可用于匹配的信息 |
| 未识别播放项 | 当前播放文件可安全展示的标题或文件名摘要 |

要求：

- 自动填充后可按当前行为自动搜索。
- 用户修改条件后可重新搜索，修改内容不得被异步回写覆盖。
- 搜索中、取消旧请求或重新发起搜索时，要有稳定状态反馈。

---

## 6. 结果列表与长字段

每条在线字幕结果可展示：

- 语言。
- release / 文件名。
- 下载量。
- 评分。
- 听障字幕标记。
- 机器 / AI 翻译标记。
- 可信上传者标记。
- FPS。
- 上传时间。
- 匹配电影或剧集信息。
- 下载按钮及下载状态。

长字段规则：

- 文件名、release 名称、匹配信息或路径摘要过长时使用省略显示。
- 鼠标悬停显示允许展示的完整内容。
- 文件路径可以展示；完整 WebDAV URL 不得裸露。
- 不展示 API key、凭据、token 或其它敏感配置内容。

---

## 7. 下载与自动切换

结果行状态：

| 状态 | 视觉与交互 |
|---|---|
| 可下载 | 显示下载按钮 |
| 下载中 | 行内 busy 状态，禁用重复点击 |
| 下载成功 | 显示成功反馈；新字幕进入在线字幕菜单并自动切换 |
| 下载失败 | 显示明确原因和可重试入口 |
| 已绑定 / 已选中 | 显示已绑定或当前字幕状态 |

下载字幕通常不需要确认弹窗。下载产生的缓存文件由缓存管理页面统一清理。

---

## 8. 绑定展示语义

以下规则仅用于 UI 归属文案、影响提示和状态展示，不扩展后端逻辑：

| 播放上下文 | 已绑定在线字幕的展示文案 |
|---|---|
| 已识别电影 | 绑定到当前电影 |
| 已识别剧集单集 | 绑定到当前集 |
| 未识别播放项 | 绑定到当前播放文件 |

设计需覆盖：

- 删除绑定时提示其影响的当前归属。
- 缓存缺失或绑定失效时说明当前字幕不可加载。
- 当前播放项修正识别后，菜单文案可以按已解析的归属重新展示，但本规格不定义新的数据迁移或后端绑定行为。
- 删除绑定只解除绑定，不删除已下载字幕缓存文件。

删除绑定确认使用播放器菜单中按钮旁的小型 Popover，详见 `player-page.md` 与 `global-dialogs.md`。

## 9. 配置、额度与错误状态

| 状态 | 提示语义 | 建议操作 |
|---|---|---|
| API 配置缺失 | 尚未配置在线字幕服务 | 提供前往设置页入口 |
| 鉴权失败 | 配置无效或登录失效 | 提示检查设置 |
| 限流 | 服务请求过于频繁 | 提示稍后重试 |
| 额度不足 | 本次下载因额度不足失败 | 提示额度原因，不预判剩余额度 |
| 下载格式不支持 | 返回文件格式无法载入 | 允许选择其它结果 |
| 压缩包无可用字幕 | 下载内容没有可用字幕文件 | 允许选择其它结果 |
| 文件过大 | 返回内容超过可处理限制 | 提示选择其它结果 |
| 网络错误 / 服务错误 | 当前请求失败 | 提供重试 |
| 搜索无结果 | 当前条件无可用结果 | 保留筛选条件并允许修改 |
| 缓存缺失 / 绑定失效 | 已绑定字幕当前无法加载 | 说明可重新下载或删除绑定 |

状态显示采用页面内 notice / error 区域，不以打断式大弹窗作为默认错误承载。

## 10. 与其它规格的关系

- 字幕菜单入口、四类分类与删除绑定轻量确认：`player-page.md`。
- 删除绑定 Popover 的统一确认语义：`global-dialogs.md`。
- 播放器与弹窗深色资源、长字段规则：`DESIGN.md` 与 `codex-ui-rules.md`。
- OpenSubtitles 配置完整规格见 `settings-page.md`；在线字幕孤立缓存保护和物理清理规格见 `cache-management-page.md`。

---

## 11. Phase 7 实施注意事项

- 优先复用现有在线字幕搜索、下载、绑定和播放切换能力。
- 不在播放器弹窗内添加配置编辑器或物理缓存清理入口。
- 不在本规格中新增字幕业务能力。
- Popup、Popover、结果行、状态提示和深色窗口控件应优先复用全局播放器控件基线。

---

## 12. 验收标准

- 可从字幕菜单的 `在线下载字幕` 二级菜单打开窗口。
- 窗口固定使用深色视觉及设计化窗口控制按钮。
- 正在播放时打开会暂停，关闭后不自动恢复播放。
- 可查看语言、类型、关键词、排序条件和结果列表。
- 下载成功后字幕自动切换并进入已绑定列表。
- API 配置、鉴权、限流、额度、格式、压缩包、大小和网络错误均有页面内状态设计。
- 已识别电影、已识别单集、未识别播放项的绑定文案分别明确。
- 删除绑定不删除缓存文件，且使用轻量确认而非居中大弹窗。
- 长字段省略与 Tooltip、WebDAV URL 脱敏规则得到遵守。
