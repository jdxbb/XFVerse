# 设置页面 UI 实施说明

## 1. 模块名称

设置

建议文件名：

- `settings-page.md`

建议实现对象：

- `SettingsPage`
- 或 `SettingsPage.xaml`

---

## 2. 页面用途

设置页面用于管理 XFVerse 的通用设置和 API 接口配置。

该页面是完整全屏页面，不是弹窗。

用户从首页左下角的头像和账号区域进入：

1. 点击首页左下角头像 / 账号区域
2. 弹出用户菜单
3. 点击菜单中的“设置”
4. 进入设置全屏页面

页面包含两个 Tab：

- 通用
- API 配置

每次进入设置页时，默认打开“通用”Tab，不需要记住上次选中的 Tab。

---

## 3. 截图说明

截图中的颜色不参与最终 UI 设计。

截图颜色只用于区分区域和信息结构。  
最终视觉必须遵循：

- `DESIGN.md`
- `resources-note.md`
- `codex-ui-rules.md`

实际 UI 应使用：

- WPF UI
- 深色 / 浅色双主题
- 灰粉 / 中性色视觉体系
- 低饱和语义色
- WPF UI 默认图标优先

## 4. 页面整体结构

设置页面从上到下包含：

1. 顶部标题区
2. Tab 区
3. 当前 Tab 内容区

```text
设置页面
├─ 顶部标题区
│  ├─ 标题：设置
│  └─ 主题切换按钮
│
├─ Tab 区
```

注意：

- 顶部不需要显示 App Logo
- 设置页右上角保留主题切换按钮
- 设置页无滚动区域
- 内容需要在全屏页面内完整展示

顶部标题区包含：

- 页面标题
- 主题切换按钮

### 页面标题

标题文本：

- `设置`

### 主题切换按钮

说明：

- 大部分页面有主题切换按钮，但不是必须全局固定

设置页有两个 Tab：

- `通用`
- `API 配置`

要求：

- 每次进入设置页默认选中“通用”

# API 配置 Tab

## 19. API 配置 Tab 结构

API 配置 Tab 包含三张配置卡片：

1. TMDB 配置
2. OMDb 配置
3. 大模型配置

结构：

```text
API 配置 Tab
├─ TMDB 配置卡片
├─ OMDb 配置卡片
└─ 大模型配置卡片
```

每张卡片包含：

- 卡片标题
- 配置状态胶囊
- 输入字段
- 测试结果
- 保存接口配置按钮
- 测试连接按钮

---

## 20. API 配置通用规则

### 输入框

API 配置中的 Token / API Key 不隐藏。

要求：

- 使用普通输入框
- 明文显示
- 不使用 PasswordBox
- 不添加显示 / 隐藏按钮
- 不使用截图中的纯黑色样式

适用字段：

- TMDB Read Access Token
- TMDB API Key
- OMDb API Key
- 大模型 Base URL
- 大模型 API Key
- 大模型模型名称

### 保存按钮

每个配置卡片单独保存自己的配置。

按钮文案：

- `保存接口配置`

要求：

- TMDB 配置卡片只保存 TMDB 配置
- OMDb 配置卡片只保存 OMDb 配置
- 大模型配置卡片只保存大模型配置
- 优先绑定现有保存配置 Command
- 不擅自修改后端业务逻辑

### 测试连接按钮

每个配置卡片都有自己的测试连接按钮。

按钮文案：

- `测试连接`

要求：

- 只测试当前配置卡片对应的接口
- 测试结果显示在当前卡片的测试结果区域
- 优先绑定现有测试连接 Command

---

## 21. API 配置状态胶囊

每个 API 配置卡片右上角都有状态胶囊。

状态胶囊包含：

- 状态灯
- 状态文字

状态文字：

- `已配置`

状态灯颜色规则：

### 保存成功后

保存接口配置成功后：

1. 显示 `已配置`
2. 自动执行一次测试连接
3. 自动测试期间状态灯显示黄色
4. 自动测试成功后状态灯显示绿色
5. 自动测试失败后状态灯显示红色

### 用户主动测试连接

用户点击 `测试连接` 时：

1. 测试期间保持测试前的状态灯颜色
2. 测试完成后根据结果更新状态灯颜色
3. 测试成功后状态灯变为绿色
4. 测试失败后状态灯变为红色

### 颜色语义

- 黄色：测试中 / 等待验证
- 绿色：测试成功 / 配置可用
- 红色：测试失败 / 配置不可用

要求：

- 颜色使用低饱和语义色
- 不使用强烈刺眼的状态色
- 状态胶囊需要适配深色 / 浅色主题

## 26. 状态处理

### 缓存清理

清理缓存时：

- 可显示按钮 loading 状态
- 清理成功后刷新对应缓存大小
- 清理失败时显示错误提示

### 保存缓存上限

保存视频缓存上限时：

- 校验输入是否合法
- 保存成功后显示轻量提示
- 保存失败时显示错误提示

### 打开缓存目录

打开缓存目录时：

- 成功则打开系统文件管理器
- 失败则显示错误提示

### API 保存

保存接口配置时：

- 保存当前卡片配置
- 保存成功后显示 `已配置`
- 保存成功后自动测试一次
- 自动测试期间状态灯为黄色
- 自动测试完成后根据结果显示绿色或红色

### API 测试

主动点击测试连接时：

- 测试期间保持测试前状态灯颜色
- 测试完成后根据结果设置绿色或红色
- 测试结果区域显示结果文本

## 27. 深色 / 浅色主题注意事项

页面视觉遵循 `DESIGN.md`。

### 浅色主题

- 页面背景为淡粉白或统一浅色背景
- 卡片使用白色或轻微渐变
- 输入框使用浅色表面
- Tab 选中态使用主色点缀
- 多选一组件选中态使用主色点缀
- 状态灯使用低饱和语义色

### 深色主题

- 页面背景使用深灰
- 卡片使用深灰表面或稍亮层级
- 输入框使用深色表面
- 使用边框和透明度区分层级
- 不使用纯黑大面积背景
- 状态灯在深色主题下仍需清晰可见
