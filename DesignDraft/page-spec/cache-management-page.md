# 软件缓存管理 UI 实施说明

## 1. 模块名称

软件缓存管理

建议承载位置：

- 设置页面 `通用` Tab 内的缓存管理 section
- 内容密度需要时，可呈现为由设置页进入的独立内容视图

关联规格：

- `settings-page.md`
- `global-dialogs.md`
- `player-page.md`
- `online-subtitle-search-page.md`

---

## 2. 页面用途

软件缓存管理只管理应用生成和维护的缓存数据，用于查看缓存占用并执行受约束的清理操作。

安全边界：

- 不删除真实本地媒体文件。
- 不删除 WebDAV 远端文件。
- 不清除用户配置。
- 不承担媒体库删除记录或移出媒体库语义。
- 在线字幕缓存只清孤立缓存，不清仍被绑定引用的缓存。

---

## 3. 最终缓存类别

Phase 6.0g-B 确认后的默认缓存卡片仅包含：

| 类别 | 用途 | 清理语义 |
| --- | --- | --- |
| 海报缓存 | 媒体海报图片缓存 | 清软件缓存，可重新拉取或生成 |
| metadata / 其它缓存 | TMDB、OMDb 等外部 metadata 缓存 | 清外部数据缓存，不清用户状态或配置 |
| 在线字幕缓存 | 播放器在线下载字幕的本地缓存 | 仅清孤立缓存，绑定引用项受保护 |

明确不提供：

- 视频缓存管理卡片
- 视频缓存大小上限配置
- 将播放器缓冲或预加载描述成离线视频缓存

---

## 4. 缓存卡片结构

每类缓存卡片包含：

- 类别名称
- 用途说明
- 当前总大小
- 当前文件数
- 可清理数量或可清理大小，如可用
- `刷新` 或统计加载状态
- `清理` 按钮
- 状态反馈区域

状态矩阵：

| 状态 | 表现 |
| --- | --- |
| loading | 统计读取中，清理操作 disabled |
| ready | 展示容量与操作 |
| empty | 显示无需清理，不突出危险操作 |
| cleaning | 当前卡片按钮 disabled，显示执行中 |
| success | 显示本次清理结果并刷新统计 |
| error | 显示安全错误提示和可重试入口 |
| disabled | 解释为什么当前不能清理 |

---

## 5. 海报缓存与 metadata / 其它缓存

### 海报缓存

- 可展示总大小、文件数和当前容量限制，如果当前能力提供该配置。
- 清理操作需经缓存清理确认。
- 文案说明海报可在后续浏览或刷新时重新加载。

### metadata / 其它缓存

- 主要表示外部 metadata 缓存。
- 不将用户状态、媒体记录或配置计入可清理内容。
- 清理操作需经缓存清理确认。

---

## 6. 在线字幕缓存

在线字幕缓存卡需要额外展示：

| 信息 | 说明 |
| --- | --- |
| 总缓存文件数 / 大小 | 当前应用缓存目录内字幕缓存概览 |
| 孤立缓存数量 / 大小 | 当前没有绑定引用，可被清理的缓存 |
| 被绑定保护数量 / 大小 | 仍被播放项绑定引用，不允许清理 |
| 清理禁用原因 | 没有孤立缓存或无法可靠判断绑定引用时显示 |

清理规则：

- `清理` 只删除孤立缓存文件。
- 仍被绑定引用的缓存必须受保护。
- 绑定读取失败或引用关系无法确认时，清理按钮 disabled，并显示安全提示。
- 播放器中删除字幕绑定只解除绑定；物理清理仍由本管理区执行。

建议安全提示：

```text
无法确认字幕绑定引用，已停止清理以避免误删正在使用的字幕缓存。
```

---

## 7. 清理确认与文案

海报缓存、metadata / 其它缓存和在线字幕缓存清理均引用 `global-dialogs.md` 的缓存清理确认体系。

普通缓存建议正文：

```text
该操作只清理软件缓存，不会删除媒体库记录、本地媒体文件、WebDAV 远端文件或用户配置。
```

在线字幕缓存建议正文：

```text
该操作仅清理未被绑定引用的在线字幕缓存，不会解除字幕绑定，也不会删除本地媒体文件或 WebDAV 远端文件。
```

按钮规则：

- 取消在左，确认清理在右。
- 确认按钮使用危险按钮样式，因为操作会移除本地应用缓存。
- disabled 状态必须给出原因。

---

## 8. 页面边界

- OpenSubtitles 凭据与默认语言配置属于 `settings-page.md`。
- 搜索、下载、切换和删除字幕绑定属于播放器链路。
- 本规格只负责缓存统计和受保护的物理清理。
- 详情页不承载在线字幕入口。

---

## 10. 验收标准

- 仅展示海报、metadata / 其它、在线字幕三类缓存。
- 不出现视频缓存管理入口或伪造的离线视频缓存进度。
- 每张卡包含容量、文件数、操作和状态反馈。
- 在线字幕卡展示孤立缓存、受保护缓存和禁用原因。
- 绑定读取失败时不允许清理在线字幕缓存。
- 清理确认明确不删除真实本地媒体文件、WebDAV 远端文件或用户配置。
- 在线字幕缓存清理不解除绑定，不清仍被绑定引用的缓存。
